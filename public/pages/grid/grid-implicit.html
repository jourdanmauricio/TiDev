Grid Implícita

Hablamos de grid explícita cuando definimos filas y columnas en el contenedor grid. Pero si no sabemos la cantidad de elementos hijos a incorporar nos referimos a grid implícitas.

Cuando incorporamos más elementos hijos a un grid que ya no posee lugar, el navegador continua apilando los elementos en la grid aunque ya no se encuentre defnida.

Si agregamos 19 elementos con una altura fija de 200px a una grid de 4x3 (12 elementos), los primeros 12 elementos conservarán su altura y se acomodarán en las 3 filas definidas. Los 7 elementos restantes se agregarán a la grid en nuevas filas pero utilizarán la altura definida por default (la altura de su contenido).

        
          .grid-implicit {
            /* Grid 4x3 */
            display: grid;
            grid-template-columns: repeat(4, 1fr);
            grid-template-rows: repeat(3, 200px);
          }
