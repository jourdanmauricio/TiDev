Tamaños Máximos y Mínimos de Grid Tracks

La función min-max define el mínimo y máximo que pueden alcanzar los tracks en el grid. Acepta dos valores: el valor mínimo que podrá tomar el track, y el máximo.

Existen dos valores especiales para esta función: min-content (se ajustará hasta el mínimo del contenido) y max-content.

        
          .grid-min-max {
            display: grid;
            /* Grid 4cols x ?  No definimos filas */
            grid-template-columns: repeat(4, minmax(100px, 150px));
            grid-template-columns: repeat(4, minmax(min-content, 150px));
            grid-template-columns: repeat(4, minmax(100px, min-content));
            grid-template-columns: repeat(4, minmax(100px, max-content));
            grid-template-columns: repeat(4, minmax(max-content, 150px));
            grid-template-columns: repeat(4, minmax(min-content, max-content));
            height: 60vh;
          }
