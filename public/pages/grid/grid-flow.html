Flujo de la Grid

El flujo natural de grid es apilar los elementos en fila, pero podemos cambiar ese comportamiento a través de la propiedad grid-auto-flow.

grid-auto-flow: row | column;

Es similar a la propiedad flex-direction de flexbox.

        
          .grid-flow {
            /* Grid 5x3 */
            display: grid;
            grid-template-columns: repeat(5, 1fr);
            grid-template-rows: repeat(3, 150px);
            grid-auto-flow: column;
            grid-auto-columns: 80px;
          }

Al utilizar grid-auto-flow: column se generan columnas implícitas.

La propiedad grid-auto-row indica como serán las filas implícitas que se generen. Su valor puede ser auto (ocupara el espacio disponible) o un valor particular. De la misma manera, existe la propiedad grid-auto-columns.

Grid dense

Para ejemplificar dense vamos a utilizar la misma grid (5 cols x 4 rows) de 19 elementos, y generaremos un desbordamiento del elemento 9 asignando 3 columnas. Como no entra en el grid, pasa a una nueva fila dejando las dos celdas de la fila anterior libres. El navegador respeta el orden de los elementos. Si alguno no entra genera una nueva fila sin adelantar otros elementos a las celdas anteriores vacias.

        
          .grid-flow-dense {
            /* Grid 5x4 */
            display: grid;
            grid-template-columns: repeat(5, 1fr);
            grid-template-rows: repeat(4, 100px);
          }
          .grid-flow-dense .item:nth-child(9) {
            color: cyan;
            grid-row: span 3;
            grid-column: span 3;
          }

Para utilizar el espacio libre que quedó en la grid podemos utilizar la propiedad grid-auto-flow: con el valor row dense

grid-auto-flow: row dense;

De esta manera, realizamos un cambio en el algorítmo del grid, indicando que posicione en el espacio libre los elementos posteriores que entren.

Tengamos en cuenta que cuando deseemos una disición por columnas en lugar de filas podemos emplear: grid-auto-flow: column dense;

        
          .grid-flow-dense {
            /* Grid 5x4 */
            display: grid;
            grid-template-columns: repeat(5, 1fr);
            grid-template-rows: repeat(4, 100px);
            grid-auto-flow: row dense;
          }
          .grid-flow-dense .item:nth-child(9) {
            color: cyan;
            grid-row: span 3;
            grid-column: span 3;
          }
