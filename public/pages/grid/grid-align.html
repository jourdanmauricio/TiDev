Alineamiento de Grid Items

La propiedad justify-items define la alineación de los elementos en la fila, en el eje x. El valor por default es strecth (que las celdas se estiren).

De la misma manera, la propiedad align-items define la alineación verticalmente (eje y) de todos los elementos.

La alineación es del item con respecto a la celda.

Cuando deseamos alinear solo un item ocupamos las propiedades justify-self y align-self.

        
          justify-items: stretch | start | end | center;
          align-items: stretch | start | end | center;
          justify-self: stretch | start | end | center;
          align-self: stretch | start | end | center;
        
      
        
          .grid-align {
            display: grid;
            grid-template-columns: repeat(3, 180px);
            grid-template-rows: repeat(2, 150px);
            height: 60vh;
            /* justify-items: stretch | start | end | center; */
            justify-items: center;
            /* align-items: stretch | start | end | center; */ 
            align-items: center;
          }
          .grid-align .item:nth-child(4) {
            justify-self: start;
            align-self: start;
          }
