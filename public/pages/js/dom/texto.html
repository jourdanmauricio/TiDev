Texto y HTML

Las propiedades innerText e textContent agregan contenido textual a un elemento html.

La propiedad innerHTML agrega el contenido textual pero reconoce e interpreta las etiquetas html.

Finalmente, la propiedad outerHTML reemplaza la etiqueta seleccionada por el contenido seleccionado.

        
          <p id="que-es">
            Lorem ipsum dolor sit amet consectetur adipisicing elit. Quidem, laboriosam cum quod, perferendis eveniet
            pariatur facere doloribus adipisci consequatur totam est dolorum vero nisi voluptate porro explicabo quo
            necessitatibus blanditiis. </p>

            const $whatIsDOM = document.getElementById( "que-es" );
            let text = `
              <p>
                El Modelo de Objetos del Documento (<b><i>DOM - Document Object Model</i></b>) es un API para documentos HTML y XML.
              </p>
              <p>
                Éste proveé una representación estructural del documento, permitiendo modificar su contenido y presentación visual mediante código JS.
              </p>
              <p>
                <mark>El DOM no es parte de la especificación de JavaScript, es una API para los navegadores.</mark>
              </p>
              `;
            // $whatIsDOM.innerText = text; 
            /* innerText no es parte del estandar, se creó para Internet Explorer y */
            /* agrega el contenido pero no reconoce las etiquetas html, las incluye como texto */
            $whatIsDOM.textContent = text;
            /* Tampoco reconoce las etiquetas html, ni los salto de línea del template */
            $whatIsDOM.innerHTML = text;
            /* innerText agrega el contenido pero reconoce e interpreta las etiquetas html */
            $whatIsDOM.outerHTML = text;
            /* outerHTML reemplaza la etiqueta seleccionada por el contenido seleccionado */
