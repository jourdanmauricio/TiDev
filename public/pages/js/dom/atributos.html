Atributos y Data-Attributes

A partir de Html5 podemos crear nuestros propios atributos que se conocen como data-attributes. El estandar indica que tenemos que utilizar la palabra clave data- y el nombre que asignemos al atributo. Ejemplo: data-description.

Existen dos formas de acceder a los atributos.

  • A través de la notación del punto
  • A través del método getAttribute('nombre-atributo')

De la misma manera, la modificación de los atribitos se puede realizar con:

  • A través de la notación del punto
  • A través del método setAttribute('nombre-atributo', 'valor')
Es recomendable utilizar la notación de corchetes por sobre la de punto.

Una práctica común al guardar elementos en variables es que el nombre de las variables relacionadas a elementos del DOM comiencen por el signo $. De esta manera, las podemos diferenciar de las varibles de nuestra lógica.

A través de hasAttribute() podemos validar si un elemento posee un atributo o eliminarlo a través utilizando removeAttribute().

        
          <a class="link-dom" data-id="1" data-description="Document Object Model" href="dom.html"
          style="background-color: #F7DF1E; color: #222;">DOM</a>

          // Enviamos a consola el atributo lang de la etiqueta html
          console.log(document.documentElement.lang); // es
          console.log(document.documentElement.getAttribute('lang')); // es
          console.log( '##################' );
          console.log(document.querySelector('.link-dom').href); // http://localhost:8080/pages/js/dom/dom.html
          console.log( document.querySelector( '.link-dom' ).getAttribute( 'href' )); // dom.html

          // Modificando el atributo
          document.documentElement.lang = 'en';
          console.log(document.documentElement.lang); // en
          document.documentElement.setAttribute('lang', 'es-AR'));
          console.log(document.documentElement.getAttribute('lang')); // es-AR

          // Guardando elemento en variables
          const $linkDOM = document.querySelector('.link-dom');
          // Modificamos el link redirigiendolo a la página principal
          // Agregamos el atributo target='_blank' para abrir la página en otra pestaña
          $linkDOM.setAttribute('href', '/index.html');
          $linkDOM.setAttribute('target', '_blank');
          $linkDOM.setAttribute('rel', 'noopener');
          console.log($linkDOM.hasAttribute('rel')); // true
          $linkDOM.removeAttribute('rel');
          console.log($linkDOM.hasAttribute('rel')); // false

          /* data-attributes */
          console.log($linkDOM.getAttribute('data-description'));
          console.log($linkDOM.dataset); 
          console.log($linkDOM.dataset.description); 
          $linkDOM.setAttribute('data-description', 'Modelo de Objeto del Documento'); 
          console.log($linkDOM.dataset.description); 
          $linkDOM.dataset.description = 'Suscribete!';
          console.log($linkDOM.hasAttribute('data-id')); // true
          $linkDOM.removeAttribute('data-id');
          console.log($linkDOM.hasAttribute('data-id')); // false
