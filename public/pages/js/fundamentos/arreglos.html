Arrays

Los arrays son objetos similares a una lista que podemos recorrer y mutar. La longitud y el tipo de los elementos de un array son variables.

Operaciones

        
          /* Crear un array */
          let frutas = ["Manzana", "Banana"];
          console.log(frutas.length);              // 2
          
          const array = [1, true, "Hola", [2, 3, 4]]
          console.log(array[3][0]);                // 2


          /* Acceder a un elemento por índice */
          let primero = frutas[0];                  // Manzana
          let ultimo = frutas[frutas.length - 1];   // Banana

          /* Recorrer un Array */
          frutas.forEach(function(elemento, indice, array) {
            console.log(elemento, indice);
          })
          // Manzana 0
          // Banana 1

          /* Añadir un elemento al final de un Array */
          let nuevaLongitud = frutas.push('Naranja'); // Añade "Naranja" al final
          // ["Manzana", "Banana", "Naranja"]

          /* Eliminar el último elemento de un Array */
          let ultimo = frutas.pop() // Elimina "Naranja" del final
          // ["Manzana", "Banana"]

          /* Añadir un elemento al principio de un Array */
          let nuevaLongitud = frutas.unshift('Fresa') // Añade "Fresa" al inicio
          // ["Fresa" ,"Manzana", "Banana"]

          /* Eliminar el primer elemento de un Array */
          let primero = frutas.shift() // Elimina "Fresa" del inicio
          // ["Manzana", "Banana"]

          /* Encontrar el índice de un elemento del Array */
          frutas.push('Fresa')
          // ["Manzana", "Banana", "Fresa"]

          let pos = frutas.indexOf('Banana') // 1

splice. Eliminar elementos

Para eliminar un elemento del array pasamos dos parámetros: la posición del primer elemento que se elimina y el número de elementos que queremos eliminar.

Con .splice() no solo se puede eliminar elementos del array, si no que también podemos extraerlos guardándolo en un nuevo array. ¡Ojo! que al hacer esto estaríamos modificando el array de origen. 
        
          let frutas = ["Manzana", "Banana", "Fresa"];

          let elementoEliminado = frutas.splice(1, 1);
          console.log(frutas); // ["Manzana", "Fresa"]

          let vegetales = ['Repollo', 'Nabo', 'Rábano', 'Zanahoria']
          console.log(vegetales)  // ["Repollo", "Nabo", "Rábano", "Zanahoria"]
          
          let elementosEliminados = vegetales.splice(1, 2);

          console.log(elementosEliminados);  // ["Nabo", "Rábano"] 
          console.log(vegetales); // ["Repollo", "Zanahoria"] 

          /* Copiar un array */
          let copiaArray = vegetales.slice();

Acceso a elementos de un array

Los índices de los arrays comienzan en cero. El índice del primer elemento de un array es 0, y el del último elemento es igual al valor de la propiedad length menos 1.

Si se utiliza un número de índice no válido, se obtendrá undefined. 

        
          let colores = ['Azul', 'Rojo', 'Amarillo']
          console.log(colores[0])                   // 'Azul'
          console.log(colores[1])                   // 'Rojo'
          console.log(colores[colores.length - 1])  // 'Amarillo'

Propiedades

  • length. Indica el número de elementos de un array.
  • filter. Retorna un nuevo array que contiene todos los elementos que cumplan el predicado que se le pasa como parámetro.
  • find. Retorna el primer elemento del array que cumpla el predicado que se pasa como parámetro, o undefined si ninguno lo cumple.
  • findIndex. Retorna el índice del primer elemento del array que cumpla el predicado que se pasa como parámetro, o -1 si nunguno lo cumple.
  • forEach. Llama a la función pasada como parámetro para todos los elementos del array.
  • includes. Determina si el array contiene el valor buscado y retorna true o false según sea el caso.
  • indexOf. Retorna el índice del primer elemento del array que sea igual al elemento buscado, o -1 si no existe.
  • join. Concatena en un string todos los elementos de un array.
  • map. Retorna un nuevo array que contiene el resultado de llamar a la función pasada como parámetro.
  • pop. Elimina el último elemento de un array, y devuelve dicho elemento.
  • push. Añade uno o más elementos al final de un array y devuelve el nuevo valor de su propiedad length.
  • reduce. Aplica la función pasada como parámetro a un acumulador y a cada valor del array, que se recorre de izquierda a derecha, para reducirlo a un único valor.
  • reverse. Invierte el orden de los elementos de un array en el propio array. Este método modifica el array.
  • shift. Elimina el primer elemento de un array, y devuelve dicho elemento.
  • slice. Extrae una porción del array sobre el que se llama y devuelve un nuevo array.
  • sort. Ordena los elementos de un array, modificando éste, y devuelve el array ordenado.
  • splice. Añade, borra o modifica elementos de un array.
  • unshift. Añada uno o más elementos al inicio de un array y devuelve el nuevo valor de length para el array resultante.
