Elementos interactivos: alert, prompt, confirm

Las 3 funciones muestran una mini ventana con el mensaje. Se denominan ventanas modales. "modal" significa que dejamos de interactuar con el resto de la página hasta que se haya ocupado de la ventana.

Función Alert

Muestra un mensaje y espera a que el usuario presione "Aceptar". Ejemplo:

        
          alert("Hello");

Función Prompt

Muestra una ventana modal con un mensaje de texto, un campo de entrada para el visitante y los botones OK/CANCELAR.

La función prompt acepta dos argumentos:

  • title: El texto a mostrar al usuario.
  • default: Un segundo parámetro opcional, el valor inicial del campo de entrada.

Podemos escribir algo en el campo de entrada de solicitud y presionar OK. Obtenemos ese texto en la variable result. O puede cancelar la entrada presionando Cancelar o presionando la tecla Esc obteniendo null en result. 

        
          let age = prompt ('¿Cuántos años tienes?', 100);

          alert(`Tienes ${age} años!`); //Tienes 100 años!

Función Confirm

La función confirm muestra una ventana modal con una pregunta y dos botones: OK y CANCELAR. El resultado es true si se pulsa OK y false en caso contrario.

         
          let isBoss = confirm("¿Eres el jefe?");
          alert( isBoss ); // true si se pulsa OK
