Operadores básicos

  • Un operando es a lo que se aplican los operadores. Por ejemplo, en la multiplicación de 5 * 2 hay dos operandos: el operando izquierdo es 5 y el operando derecho es 2.
  • Un operador es unario si tiene un solo operando. Por ejemplo, la negación unaria - invierte el signo de un número
    •           
            let x = 1;
            x = -x;
            alert( x ); // -1, se aplicó negación unaria
  • Un operador es binario si tiene dos operandos. El mismo negativo también existe en forma binaria 
    •           
            let x = 1, y = 3;
            alert( y - x ); // 2, binario negativo resta valores

    Estamos hablando de dos operadores distintos: la negación unaria (un operando: revierte el símbolo) y la resta binaria (dos operandos: resta).

Operadores matemáticos

  • Suma +
  • Resta -
  • Multiplicación *
  • División /
  • Resto %
  • Exponenciación **

Resto %

El resultado de a % b es el resto de la división entera de a por b.

        
          alert( 5 % 2 ); // 1 es un resto de 5 dividido por 2
          alert( 8 % 3 ); // 2 es un resto de 8 dividid

Exponenciación **

El operador exponenciación a ** b eleva a a la potencia de b. 

        
          alert( 2 ** 2 ); // 2² = 4
          alert( 2 ** 3 ); // 2³ = 8
          alert( 2 ** 4 ); // 2⁴ = 16

Matemáticamente, la exponenciación es definida para números no enteros también. Por ejemplo, la raíz cuadrada es el exponente ½:

        
          alert( 4 ** (1/2) ); // 2 (potencia de 1/2 es lo mismo que raíz cuadrada)
          alert( 8 ** (1/3) ); // 2 (potencia de 1/3 es lo mismo que raíz cúbica)

Concatenación de cadenas con el binario +

Si aplicamos el + binario a cadenas, las une (concatena):

        
          let s = "my" + "string";
          alert(s); // mystring

El binario + es el único operador que soporta cadenas en esa forma. Otros operadores matemáticos trabajan solamente con números y siempre convierten sus operandos a números. Por ejemplo, resta y división:

        
          alert( 2 - '1' ); // 1
          alert( '6' / '2' ); // 3

Modificar en el lugar

A menudo necesitamos aplicar un operador a una variable y guardar el nuevo resultado en esa misma variable. 

        
          let n = 2;
          n = n + 5;
          n = n * 2;

          // Podemos utilizar los operadores += y -=
          n += 5; // ahora n = 7 (lo mismo que n = n + 5)
          n *= 2; // ahora n = 14 (lo mismo que n = n * 2)

          alert( n ); // 14

Incremento/decremento

Aumentar o disminuir un número en uno es una de las operaciones numéricas más comunes. Puedes utilizar los operadores ++ y --

        
          let counter = 2;
          counter++;      // funciona igual que counter = counter + 1, pero es más corto
          alert( counter ); // 3

          ////////////////
          
          counter = 2;
          counter--;      // funciona igual que counter = counter - 1, pero es más corto
          alert( counter ); // 1

Los operadores ++ y -- pueden ser colocados antes o después de una variable.

  • Cuando el operador va después de la variable, está en “forma de sufijo”: counter++.
  • La “forma de prefijo” es cuando el operador va antes de la variable: ++counter.

La forma prefijo devuelve el nuevo valor mientras que la forma sufijo devuelve el valor anterior (antes del incremento/decremento).

        
          let counter = 1;
          let a = ++counter; 

          alert(a); // 2

          ////////////////////

          counter = 1;
          let a = counter++; 

          alert(a); //  1
