Objetos Console, Date y Math

Objeto console

El objeto Console provee acceso a la consola de depuración de los navegadores (ejemplo, la Consola Web en Firefox o chrome). Los detalles de como funciona varían de navegador en navegador, pero hay un conjunto de características que de facto son generalmente proporcionadas.

Durante esta guía ya hemos utilizado el objeto console. Como aún no hemos interactuado con el DOM, estamos utilizando console para mostrar os resultados.

Métodos

  • Console.assert(). Registra un mensaje y envía una traza de error a la consola si el primer argumento es false.
  • Console.count(). Registra el numero de veces que esta línea ha sido llamada con etiqueta dada.
  • Console.error(). Muestra un mensaje de error. Se pueden utilizar sustituciones de cadena y argumentos adicionales con este método.
  • Console._exception(). Es un alias de error().
  • Console.group(). Crea un nuevo grupo, indentando todos los mensajes subsecuentes en un nuevo nivel. Para retroceder un nivel, se utiliza groupEnd().
  • Console.groupCollapsed(). Crea un nuevo grupo, indentando todos los mensajes subsecuentes en un nuevo nivel. A diferencia de group(), inicia con la línea de grupo colapsada, requiriendo el uso de un botón de apertura para expandir el grupo. Para retroceder un nivel, se utiliza groupEnd().
  • Console.groupEnd(). Finaliza el grupo actual.
  • Console.info(). Muestra un mensaje de información en la Consola Web. Se puede utilizar sustitución de cadena y argumentos adicionales con este método.
  • Console.log(). Para mensajes generales de registro de información. Se puede utilizar sustitución de cadena y argumentos adicionales con este método.
  • Console.table(). Muestra datos tabulares en forma de una tabla.
  • Console.time(). Inicia un temporizador con un nombre específico como parámetro. Hasta 10.000 temporizadores simultáneos pueden ejecutarse en una página determinada.
  • Console.timeEnd(). Detiene el temporizador especificado y registra el tiempo transcurrido, en segundos, desde su inicio.
  • Console.warn(). Muestra un mensaje de advertencia. Se puede utilizar sustitución de cadena y argumentos adicionales con este método.

Enviar texto a la consola

La característica mas utilizada de la consola es la de mostrar texto y otro datos. Existen cuatro categorías de salida que se pueden generar, utilizando los métodos console.log(), console.info(), console.warn() y console.error(). Cada uno de estos muestran resultados que se lucen diferentes en el registro y se pueden utilizar los controles de filtro proveidos por el navegador para ver únicamente los tipos de salida de interés.

La manera mas simple de utilizar los métodos de registro es mostrar un texto u objeto.

          
            let nombre = "Mauricio"; 
            let apellido = "Jourdan";

            console.log(nombre, apellido);
            // Interpolación
            console.log( `Hola. Mi nombre es: ${nombre}` );
            console.log("Este es un texto");
            console.error("Este es un error");
            console.warn("Este es una advertencia");
            console.info("Este es un mensaje de información");

Utilizando sustituciones de cadena

          
            for (var i=0; i<5; i++) {
              console.log('Hola, %s. Usted me ha llamado %d veces.', 'Bob', i+1);
            }

            // Hola, Bob. Usted me ha llamado 1 veces.
            // Hola, Bob. Usted me ha llamado 2 veces.
            // Hola, Bob. Usted me ha llamado 3 veces.
            // Hola, Bob. Usted me ha llamado 4 veces.
            // Hola, Bob. Usted me ha llamado 5 veces.

Dando estilo a la salida de la consola.

Puedes usar la directiva "%c" para aplicar un estilo CSS a la salida de la consola

          
            console.log("%cMy stylish message", "color: red; font-style: italic");

Usando grupos en la consola

Puedes usar grupos anidados para organizar la salida. Para crear un nuevo bloque anidado se utiliza console.group(). El método console.groupCollapsed() es similar, pero requiere el uso de un "botón de mostrar" para abrirlo.

          
            console.log("Fuera del grupo");
            console.group();
            console.log("Nivel 2");
            console.group();
            console.log("Nivel 3");
            console.warn("Más en nivel 3");
            console.groupEnd();
            console.log("De vuelta en nivel 2");
            console.groupEnd();
            console.debug("Fuera del grupo");

Usando tablas en la consola

Muestra datos tabulares como una tabla. Esta función toma un argumento obligatorio: data, que debe ser un array o un objeto, y un parámetro adicional: columns. 

          
            const numeros = [1, 2, 3, 4, 5]
            const vocales = ['a', 'e', 'i', 'o', 'u']
            const gato = {
              nombre: "Hermes",
              raza: "Siames",
              edad: 7
            }
  
            console.table( numeros );
            console.table( vocales );
            console.table( gato );

Timers

Podeoms calcular la duración de una operación específica iniciando un contador (método console.time()) pasándole como parámetro un nombre. Para detener el contador, y obtener el tiempo transcurrido en milisegúndos, se usa el método console.timeEnd(), nuevamente pasándole el nombre del contador como parámetro. 

          

            console.time("Temporizando");

            const arreglo = Array(1000000);
            
            for (let i = 0; i < arreglo.legth; i++) {
              arreglo[i] = i;
            }

            console.timeEnd("Temporizando");

Mostrando cuántas veces se ejecuta un bloque. console.count()

Registra el número de veces que se llama a count(). Esta función toma como argumento opcional una etiqueta.

Si la etiqueta es colocada, esta funcion muestra el numeros de veces que count() ha sido utilizada con la misma etiqueta. Si la etiqueta es omitida, la funcion registra el numero de veces que count() fue llamada en esta línea en particular.

          
            let user = "";

            function greet() {
              console.count(user);
              return "hi " + user;
            }
            
            user = "Juan";
            greet();
            user = "Alicia";
            greet();
            greet();
            console.count("Alicia");

Haciendo pruebas con assert

En JavaScript existen varias librerías para realizar prubas unitarias, pero el método console.assert() nos permite realizar pequeñas pruebas utilizando la consola.

Parámetros

  • Condición. Cualquier expresión booleana.Si la afirmación es falsa, el mensaje aparecerá en consola.
  • obj1 ... objN. Salida de una lista de objetos de JavaScript. Las representaciones de cadena de cada uno de estos objetos se añaden juntos en el orden y salida indicados.
  • msg. Una cadena de JavaScript que contiene cero o más cadenas de sustitución.
  • subst1 ... substN. Objetos de JavaScript con el que reemplaza las cadenas dentro de msg. Esto sno dará más control sobre el formato de salida.
          
            let x = 1;
            let y = 2;
            let pruebaXY = "esperamos que x siempre sea menor que y";
  
            console.assert( x < y, { x, y, pruebaXY } );
  
            x = 3;
            console.assert( x < y, { x, y, pruebaXY } );

Objeto Date

Los objetos Date representan en JavaScript un momento fijo en el tiempo en un formato independiente. El objeto Date contiene un Number que representa los milisegundos transcurridos desde el 1 de Enero de 1970 UTC. 

          
            console.log(Date()); // Sun Jun 05 2022 17:30:15 GMT-0300 (hora estándar de Argentina)

Hay muchos métodos disponibles para obtener una fecha en distintos formatos, también para representar conversiones de zona horaria. Son particularmente útiles las funciones que retornan la fecha y la hora en Tiempo Universal Coordinado (UTC), la hora estandar global definida por la Hora Mundial Estandar. Esta hora tambien es conocida como Hora media de Greenwich.

Constructor
Date(). Cuando es llamado como una función, retorna una cadena que representa la fecha y hora actual, exactamente como lo hace new Date().toString().
new Date(). Cuando es llamado como un constructor, retorna un nuevo objeto Date.

Métodos estáticos

  • Date.now(). Devuelve el valor númerico correspondiente al actual número de milisegundos transcurridos desde el 1 de Enero de 1970, 00:00:00 UTC, ignorando los segundos intercalares.
  • Date.parse(). Transforma la cadena que representa una fecha y retorna el número de milisegundos transcurridos desde el 1 de Enero de 1970, 00:00:00 UTC, ignorando los segundos intercalares.
  • Date.UTC(). Acepta los mismos parámetros de la forma extendida del constructor (por ejemplo: del 2 al 7 ) y retorna el número de milisegundos transcurridos desde el 1 de Enero de 1970, 00:00:00 UTC, ignorando los segundos intercalares.
          
            let fecha = new Date();
            console.log( fecha );
  
            // Número de día del mes
            console.log( 'Número de día: ' + fecha.getDate() );
            // Día de la semana. Van de domingo (0) a sábado (6).
            console.log( 'Día de la semana: ' + fecha.getDay() );
            // Mes. Enero es 0, diciembre 11.
            console.log( 'Mes: ' + fecha.getMonth() );
            // Año.
            console.log( 'Año: ' + fecha.getFullYear() );
            // Hora - Munitos - Segundos - Milisegundos
            console.log( 'Hora: ' + fecha.getHours() + '-' + fecha.getMinutes() + '-' + fecha.getSeconds() + '-' + fecha.getMilliseconds() );
            // toString
            console.log( 'toString:' + fecha.toString() );
            // Solo la Fecha 
            console.log( fecha.toDateString() );
            // toLocaleString
            console.log( 'toLocaleString: ' + fecha.toLocaleString() );
            // toLocaleDateString
            console.log( 'toLocaleDateString: ' + fecha.toLocaleDateString() );
            // toLocaleTimeString
            console.log( 'toLocaleTimeString: ' + fecha.toLocaleTimeString() );
            // UTC. Fecha y hora mundial
            console.log( 'getUTCDate: ' + fecha.getUTCDate() );
            console.log( 'getUTCHours: ' + fecha.getUTCHours() );
            // now. Timestamp. Generalmente utilizado en BD, milisegundos desde el 1 de Enero de 1970. 
            console.log( 'now: ' + Date.now() );
Si nuestra aplicación se ejecuta en un navegador (cliente), los métodos mostrarán la fecha y hora local por más que nuestra web se aloje en un servidor desde Alemania.

Para realizar cálculos entre fechas podemos utilizar el formato Timestamp y luego formatear al tipo de visualización que deseemos.

Una librería muy utilizada es moment.js

Objeto Math

Math es un objeto incorporado que tiene propiedades y métodos para constantes y funciones matemáticas. No es un objeto de función. Math funciona con el tipo Number. No funciona con BigInt.

Existen muchas porpiedades y métodos en el objeto Math, los podemos ver en la documentación de MDN. También los podemos ver ejecutando console.log(Math);

          
            console.log('PI: ', Math.PI);
            // Valor absoluto de un número
            console.log('Abs: ', Math.abs(-7.8));
            // Redondeo
            console.log('Ceil: ', Math.ceil(7.5));
            console.log('Floor: ', Math.floor(7.5));
            console.log('Round: ', Math.round(7.2));
            console.log('Round: ', Math.round(7.8));
            // Raiz cuadrada
            console.log('Sqtr: ', Math.sqtr(81));
            // Pontencias
            console.log('Pow: ', Math.pow(2, 5));
            // Signo
            console.log('Sign: ', Math.sign(-7.8));
            console.log('Sign: ', Math.sign(7.8));
            console.log('Sign: ', Math.sign(0));
            // Ramdom. Valores aleatorios entre 0 y 1
            console.log('Ramdom: ', Math.random());
            // Ramdom. Valores aleatorios entre 0 y 1000
            console.log('Ramdom: ', math.round( Math.random() * 1000 ) );
