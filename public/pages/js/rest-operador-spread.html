Rest y spread operator

Parámetros Rest

La sintaxis de los parámetros rest nos permiten representar un número indefinido de argumentos como un array.

El último parámetro de una función se puede prefijar con ..., lo que hará que todos los argumentos restantes (suministrados por el usuario) se coloquen dentro de un array. Sólo el último parámetro puede ser un "parámetro rest". 

        
          function sumar(a, b, ...c) {
            let resultado = a + b;
            
            c.forEach(function(n)) {
              resultado += n;   
            }
            return resultado;
          }

          console.log(sumar(1,2,3,4)); // 10

Operador de propagación (spread operator)

La sintaxis spread operator permite a un elemento iterable tal como un arreglo, una cadena o un objeto ser expandido.

          
            const arr1 = [1, 2, 3, 4, 5];
            const arr2 = [6, 7, 8, 9, 0];

            const arr3 = [...arr1, ...arr2];

            console.log(arr3);  // [1, 2, 3, 4, 5, 6, 7, 8, 9, 0]
