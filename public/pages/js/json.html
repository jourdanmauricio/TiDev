Json (JavaScript Object Notation)

Json es un formato ligero de intercambio de datos. Se ha convertido en un standard.

JSON es una sintaxis para serializar objetos, arreglos, números, cadenas, booleanos y nulos. Está basado sobre sintaxis JavaScript.

Supongamos que trabajamos sobre el sistema de pago en un sitio de comercio electrónico. Al momento de procesar el pago de los productos tendremos que enviar la información a la pasarela de pagos. Tendremos que enviar el nro de tarjeta, titular, monto, etc.

Entonces para realizar la comunicación entre los distintos sistemas utilizamos el formato Json, aunque algunos sistemas utilizan otros formatos como XML.

En la documentación de Json podemos ver los tipos de datos soportados y los lenguajes que implementan el soporte para este formato.

Los archivos .json deben comenzar y terminar con llaves { }, y las propiedades se escriben entre comillas dobles "". 

      
        // Ejemplo Json en javascript
        const json = {
          cadena: "Jon",
          numero: "35",
          booleano: "true",
          arreglo: ["correr", "programar", "cocinar"],
          objeto: {
            twitter: "@jonmircha",
            email: "jonmircha@gmail.com"
          },
          nulo: null
        }

JSON.parse() y JSON.stringify()

JSON.parse() analiza una cadena de texto JSON, y la transforma a un tipo de dato que JavaScript valide, como un objeto, una cadena, un booleano, etc.

JSON.stringify() toma un objeto o valor valido en JavaScript en una cadena de texto Json.

      
        // transformamos cadenas de texto en objetos, arrays, ect.
        console.log(JSON.parse("{}")); // {}
        console.log(JSON.parse("[1, 2, 3]")); // [1, 2, 3]
        console.log(JSON.parse("true")); // true
        console.log(JSON.parse("false") === false); // true

        console.log(JSON.stringify({})); // '{}'
        console.log(JSON.stringify([1, 2, 3])); // '[1, 2, 3]'
        console.log(JSON.stringify(true)); // 'true'
        console.log(JSON.stringify(false) === 'false'); // true
        console.log(JSON.stringify({x:2, y:3})) // '{"x":2, "y":3}'
