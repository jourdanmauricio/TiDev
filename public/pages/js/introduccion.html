Introducción

JavaScript es un lenguaje de programación de scripts (secuencia de comandos) orientado a objetos.

JavaScript es un lenguaje interpretado y permite programar scripts. El interpretador para nustro código será el navegador de internet.

Javascript añade interacción al usuario, animaciones, navegación, etc. Inicialmente era un lenguaje del lado del cliente, es decir que los scripts son ejecutados por el navegador del usuario (cliente). Esto difiere de los llamados lenguajes de script del lado del servidor que son ejecutadas por el servidor web.

Node js, es un entorno en tiempo de ejecución multiplataforma para la capa del servidor (en el lado del servidor) basado en JavaScript. Node js nació cuando los desarrolladores de JavaScript lo extendieron de la ejecución en el navegador a algo que se puede ejecutar en una máquina como una aplicación independiente.

El propósito de los scripts del lado del cliente y del lado del servidor es distinto. Un script del lado del servidor se encargará de "crear" la página web que será enviada al navegador (cliente), que mostrará la página ejecutando la secuencias de comandos.

Javascript es un lenguaje con muchas posibilidades, permite la programación de pequeños scripts, pero también de programas más grandes, orientados a objetos, con funciones, estructuras de datos complejas, etc.

Características

  • Lenguaje de alto nivel
  • Interpretado
  • Dinámico
  • Debilmente tipado
  • Multi paradigma
  • Sensible a mayúsculas y minúsculas
  • No requiere punto y coma al final de cada línea
