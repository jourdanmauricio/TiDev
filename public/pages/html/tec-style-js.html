Estilos CSS en Html

Existen tres formas de aplicar estilos CSS:

  • Etiqueta style dentro del elemento HTML. En el siguiente ejemplo se aplica un fondo amarillo y color de letra roja: 
                  
                <div style="background-color:yellow; color:red"> 
                  Este es un bloque DIV. Pintamos el fondo de amarillo para que se vea su magnitud real. Vemos como 
                  ocupa el 100% del ancho aunque no lo necesite. 
                </div>
    Este es un bloque DIV. Pintamos el fondo de amarillo para que se vea su magnitud real. Vemos como ocupa el 100% del ancho aunque no lo necesite.
  • Etiqueta style dentro de la cabecera HEAD del documento HTML. El siguiente ejemplo pintará todos los títulos de tipo h2 de color azul: 
                  
                <!DOCTYPE html>
                <html lang="en">
                  <head>
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                    <style>
                      h2 {
                        color: blue;
                      }
                    </style>
                  </head>
                  <body>
                  </body>
                </html>
  • Utilizando una hoja de estilos. Es la forma correcta de incluir los estilos. En el HEAD linkeamos el archivo con extensión .css que contiene los estilos, y ya estarán disponibles para utilizarlos. Utilizamos la etiqueta link. De esta manera separamos el html de CSS 
                  
                <!DOCTYPE html>
                  <html lang="en">
                  <head>
                    <meta charset="UTF-8">
                    <meta http-equiv="X-UA-Compatible" content="IE=edge">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Document</title>
                    <link rel="stylesheet" href="style.css">
                  </head>
                  <body>
                  </body>
                </html>
    Y el archivo style.css: 
                  
                h1 {
                  color: red;
                }

Scritps en html

Existen dos formas de incluir scripts en nuestra página web.

  • Mediante la etiqueta script: se ubica al final del documento antes de cerrar la etiqueta BODY. El siguiente ejemplo muestra un mensaje a través de la consola para desarrollo. Lo importante es ver cómo incluir los scripts, el contenido lo veremos en la sección JavaScript. 
                  
                <html>
                ...
                  <body>
                   ...
                   <script>console.log("Hola, inprimimos en la consola!")</script>
                  </body>
                </html>
  • Lo ideal es que la programación javascript se encuentre en archivos separados. Podemos vincularlos utilizando el atributo src de la etiqueta SCRIPT: 
                  
                <html>
                ...
                  <body>
                    ...
                    <script src="codigo.js"></script>
                  </body>
                </html>
