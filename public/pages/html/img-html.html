Agregando imágenes

La etiqueta para agregar una imagen es IMG

Imágenes: jpg, png, gif, svg

            
              <img src="" alt="">

El atributo source (scr) contiene la ruta (path) y el nombre de archivo de la imágen. El contenido del atributo alt es texto que se mostrará en el caso que el navegador tenga algún problema al intentar renderizar la imágen. Es importante incluir un texto en el atributo alt para los screen readers (software para personas con discapacidad visual). Por otro lado, los robots también utilizan esta información para el SEO.

Los formatos más populares soportados por los navegadores web son: .jpg, .png, .gif animado, y el .svg. Incluiremos una imágen de cada uno de los tipos

            
              <img src="/assets/images/imagen-jpg.jpg" height="200"  width="200" alt="Imágen de pinguino en formato jpg">
              <img src="/assets/images/imagen-png.png" height="200"  width="200" alt="Imágen de pinguino en formato png">
              <img src="/assets/images/imagen-gif.gif" height="200"  width="200" alt="Imágen de pinguino en formato gif">
              <img src="/assets/images/imagen-svg.svg" height="200"  width="200" alt="Imágen de pinguino en formato svg">

La etiqueta img admite los atributos height y width para indicar alto y ancho de la imágen en pixeles. No es recomendable definir los tamaños con estos atributos. Lo ideal es realizarlo a través de css

Podemos agregar una regla css que indique:

            
              img {
                max-width: 100%; /*aunque la imagen sea más grande se reducirá al ancho de pantalla*/
                          height: auto: /*de esta manra no se deformará. Mantiene la relación alto-ancho*/
              }

Vectores

Como vimos, podemos agregar los vectores svg a etiquetas img, pero son código xml. Quiere decir que podemos incluir el código directamente en el html. Una característica de los svg es que podemos redimensionarlos sin perder calidad. Se utiliza mucho en logos

Existen muchas páginas para descargar íconos vectioriales, como: https://css.gg o https://www.svgrepo.com

Etiqueta figure

Es una etqueta semántica. Nos permite agregar imágenes, videos, etc con una nota al pie (es opcional)

          
            <figure>
              <img src="/assets/images/table_elements_tags.png" alt="Tabla de elementos html">
              <figcaption>Tabla de elementos html</figcaption>
            </figure>
