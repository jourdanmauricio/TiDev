Listas

Existen 3 tipos de listas, aunque generalmente utilizamos las ordenadas y las desordenadas.

Listas ordenadas

Es una lista numerada y se representa con la etiqueta OL (order list). Cada elemento se debe incorporar con una etiqueta li (list item)

          
            <h4>Ejemplo: Estaciones del año</h4>
            <ol>
              <li>Primavera</li>
              <li>Verano</li>
              <li>Otoño</li>
              <li>Invierno</li>
            </ol>

Ejemplo: Estaciones del año

  1. Primavera
  2. Verano
  3. Otoño
  4. Invierno

Si queremos comenzar con una numeración diferente podemos agregar el atributo start="3". También podemos invertir la lista con el atributo booleano reversed. Finalmente, el atributo type nos permite seleccionar el tipo de mumeración:

  • Números (type="1")
  • Letras mayúsculas (type="A")
  • Letras minúsculas (type="a")
  • Números romanos en mayúsculas (type="I")
  • Números romanos en minúsculas (type="i")
          
            <h4>Ejemplo: Estaciones del año</h4>
            <ol start="4" type="A" reversed>
              <li>Primavera</li>
              <li>Verano</li>
              <li>Otoño</li>
              <li>Invierno</li>
            </ol>

Ejemplo: Estaciones del año

  1. Primavera
  2. Verano
  3. Otoño
  4. Invierno

Listas desordenadas

Son las viñetas. Se representan a través de la etiqueta ul (unorder list)

Podemos utilizar el atributo type con los valores disc (predeterminado), circle, square

          
            <h4>Cosas para hacer</h4>
            <ul>
              <li>Estudiar inglés</li>
              <li>Preparar presentación del proyecto</li>
              <li>Compartir la presentación</li>
            </ul>
            <ul type="circle">
              <li>Estudiar inglés</li>
              <li>Preparar presentación del proyecto</li>
              <li>Compartir la presentación<li>
            </ul>
            <ul type="square">
              <li>Estudiar inglés</li>
              <li>Preparar presentación del proyecto</li>
              <li>Compartir la presentación</li>
            </ul>

  • Preparar presentación del proyecto
  • Compartir la presentación

Listas de definición

Se utiliza para glosario de términos. Se representan con la etiqueta dl (definition list), la etiqueta dt (definition term) para el término que queremos explicar y para la definicion utilizamos dd (definiton data) 

          
            <dl>
              <dt>HTML</dt>
              <dd>Es un lenguaje de marcado para definir el contenido de nuetsra web</dd>
              <dt>CSS</dt>
              <dd>Es un lenguaje de definición que da estilos al código html</dd>
              <dt>JavaScript</dt>
              <dd>Es un lenguaje de la Web</dd>
            </dl>
