Iframes

La recomendación es no utilizar este recurso. Es conveniente utilizar javascript con ajax, pero si queremos incrustar un elemento de otra página o un pdf podemos utilizarla aunque podemos tener problemas a la hora de cargar la página.

Hay páginas que se encuentran protegidas para no ser incrutadas.

        
          <iframe src="/index.html"></iframe>

Google maps y youtube nos permiten incorporar elementos a través de esta etiqueta. Ejemplo:

        
          <iframe src="https://www.google.com/maps/embed?pb=!1m18!1m12!1m3!1d3284.0167132768483!2d-58.38375908524271!3d-34.60373888045943!2m3!1f0!2f0!3f0!3m2!1i1024!2i768!4f13.1!3m3!1m2!1s0x4aa9f0a6da5edb%3A0x11bead4e234e558b!2sObelisco!5e0!3m2!1sen!2sar!4v1651524263141!5m2!1sen!2sar" width="600" height="450" style="border:0;" allowfullscreen="" loading="lazy" referrerpolicy="no-referrer-when-downgrade"></iframe>
