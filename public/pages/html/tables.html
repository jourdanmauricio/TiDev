Tablas

A finales de los 90 las tablas se utilizaban para maquetar. No existían sistemas como grid o flexbox (los veremos en la sección CSS). Se representa a con la etiqueta table. Las tablas contienen filas y columnas. Las filas se definen con la etiqueta tr (table row) y las celdas que contiene la fila se define con td (table data)

        
          <table>
            <tr>
              <td>Banda</td>
              <td>Tipo</td>
              <td>Año de formación</td>
            </tr>
            <tr>
              <td>Queen</td>
              <td>Rock / Pop</td>
              <td>1970</td>
            </tr>
            <tr>
              <td>Metallica</td>
              <td>Thrash metal</td>
              <td>1983</td>
            </tr>
            <tr>
              <td>AC/DC</td>
              <td>Rock</td>
              <td>1973</td>
            </tr>
          </table>
Banda Tipo Año de formación

Podemos agrupar las filas a través de 3 eqtiquetas: thead, tbody y tfoot. En caso de utilizar thead, la etiqueta tr se reemplaza por th. También podemos combinar filas y columnas con las etiquetas colspan y rwospan 

        
          <table>
            <thead>
              <tr>
                <th colspan="3">Tabla de bandas históricas</th>
              </tr>
              <tr>
                <th>Banda</th>
                <th>Tipo</th>
                <th>Año de formación</th>
              </tr>
            </thead>
            <tbody>
              <tr>
                <td>Queen</td>
                <td>Rock / Pop</td>
                <td>1970</td>
              </tr>
              <tr>
                <td>Metallica</td>
                <td>Thrash metal</td>
                <td>1983</td>
              </tr>
              <tr>
                <td>AC/DC</td>
                <td rowspan="2">Rock</td>
                <td>1973</td>
              </tr>
              <tr>
                <td>Led Zeppelin</td>
                <td>1968</td>
              </tr>
            </tbody>
            <tfoot>
              <tr>
                <th colspan="3"><small>La historia del Rock</small></th>
              </tr>
            </tfoot>
          </table>
