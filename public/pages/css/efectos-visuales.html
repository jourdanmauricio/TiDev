Efectos visuales en CSS

Sombras

Podemos aplicar tres tipos de sombras: al texto, a la caja y existe un filtro llamado drop-shadow.

box-shadow

Define la sombra del elemento.

box-shadow: mov-x mov-y radio spread(propagación) color inset/outset(interna/externa)

  • box-shadow: none; Elimina cualquier sombra de cuadro que se haya aplicado al elemento
  • box-shadow: 2px 6px; Necesitamos al menos dos valores: el primero es el desplazamiento horizontal el segundo es el desplazamiento vertical. El color de la sombra se heredará del color del texto
  • box-shadow: 2px 6px red; Puede definir un color como último valor. Al igual que con el color, puede usar nombres de color, hexadecimal, rgb, hsl...
  • box-shadow: 2px 4px 10px red; El tercer valor opcional define el desenfoque de la sombra. El color se difundirá a través de 10px en este ejemplo, de opaco a transparente
  • box-shadow: 2px 4px 10px 4px red; El cuarto valor opcional define la extensión de la sombra. La extensión define cuánto debe crecer la sombra: realza la sombra

text-shadow

Define la sombra del contenido del texto.

  • text-shadow: none; El contenido del texto no tiene sombra
  • text-shadow: 2px 6px; Necesitamos al menos dos valores: el primero es el desplazamiento horizontal el segundo es el desplazamiento vertical El color de la sombra se heredará del color del texto
  • text-shadow: 2px 6px red; Podemos definir un color como último valor. Al igual que con el color, puede usar nombres de color, hexadecimal, rgb, hsl...
  • text-shadow: 2px 4px 10px red; El tercer valor opcional define el desenfoque de la sombra. El color se difundirá a través de 10px en este ejemplo, de opaco a transparente

              width: 200px;
              height: 200px;
              border: thick solid black;
              margin-left: auto;
              margin-right: auto;
            }
            .box-shadow {
              color: navy;
              font-size: 2rem;
              text-align: center;
              box-shadow: 1rem 1rem 1rem 1rem black;
              box-shadow: -1rem -1rem 1rem 1rem black;
              box-shadow: 1rem 1rem 1rem 1rem #0005;
              box-shadow: 1rem 1rem 1rem 1rem #0005 inset;
              box-shadow: 1rem 1rem 1rem 1rem #0005;
              text-shadow: 1rem 1rem 0.3rem hotpink;
            }

Propiedad filter

La propiedad CSS filter dota de efectos gráficos como el desenfoque o cambio de color en la representación antes de que se muestre el elemento. Los filtros se utilizan comúnmente para ajustar la representación de imágenes, fondos o bordes.

Para utilizar la propiedad CSS filter, hay que especificar un valor para una de las siguientes funciones. Si el valor no es válido, la función se ejecutará de la misma manera que si se le aplicara "none"

Función drop-shadow

Aplica un efecto de sombra a la imagen. Una sombra es realmente una versión desenfocada y separada de la máscara alfa de la imagen, dibujada en un color particular, debajo de la misma. La función acepta: mov-x mov-y blur-radius color.

Como vemos la sombra aplica sobre el contenido de la imágen, pero debe tener transparencia.

          
            <div class="box drop-shadow">
              <img src="/assets/images/arbol.png" alt="Niño y arbol">
            </div>
          
        
          
            .drop-shadow {
              width: 600px;
              height: auto;
            }
            .drop-shadow img {
              max-width: 100%;
              height: auto;
              box-shadow: 1rem 1rem 1rem 1rem #f00;
              filter: drop-shadow(1rem 1rem 1rem orangered);
              filter: drop-shadow(1rem 1rem 1rem skyblue);
            }

Degradados

Los gradientes trabajan sobre la propiedad backgorund pero es una buena práctica realizarlo sobre la propiedad background-image.

Lineales

La función linear-gradient admite los siguientes parámetros.

  • linear-gradient(color, color, ...); Gradiente de colores dirección hacia abajo
  • linear-gradient(direction, color, color); Gradiente con dirección específica
  • linear-gradient(direction, color, size, color, size); Gradiente indicando donde comienza a cambiar el color
  • linear-gradient(direction, color, size, size, color, size, size); Gradiente indicando inicio y fin de cada color

La dirección se puede expresar a través de palabras clave o de grados.

Podemos definir la proporción de cada uno de los colores asignado porcentajes

Dirección Grados
to top 0 o 360deg
to top right 45deg
to right 90deg
to bottom right 145deg
to bottom 180deg
to bottom left 215deg
to left 270deg
to top left 325deg

Las distancias entre colores se ajustan automáticamente con el mismo tamaño. Pero, podemos asignar un tamaño justo después del color en cuestión.

También podemos definir un punto de inicio y de fin para cada color.

          
            <div class="box linear-gradient"></div>
          
        
          
            .linear-gradient {
              background-image: linear-gradient(red, green);
              background-image: linear-gradient(20deg, red, green, blue);
              background-image: linear-gradient(45deg, red, green, blue);
              background-image: linear-gradient(to top left, red, green, blue);
              background-image: linear-gradient(90deg, red 30%, green 31% 60%, blue 61%);
              background-image: linear-gradient(90deg, red 20%, green 60%, blue);
            }

Radiales

El gradiente radial nos permite crear degradados con formas circulares. El formato para los gradientes radiales es muy parecido al anterior. En este caso, debemos usar la función radial-gradient() y un nuevo atributo para la forma.

  • background: radial-gradient(forma, color, color, color...);
  • background: radial-gradient(forma tamaño, color, color, color...);
  • background: radial-gradient(forma, color tamaño, color tamaño, color tamaño...);
  • background: radial-gradient(color, color, color...);

Existen dos valores para la forma:

forma: existen dos tipos de forma: circle y ellipse (por defecto ellipse). Opcionalmente podemos indicar el tamaño de la forma.

color: permite crear una lista para ir cambiando el color del gradiente. Podemos definir tantos colores como deseemos.

tamaño: para cada color se puede indicar la posición en la que comienzan las transiciones.

                circle 100px,

                yellow 90%

              background-image: radial-gradient(
                circle 100px at top,

Filtros

La propiedad filter permite efectos gráficos como el desenfoque o cambio de color en la representación antes de que se muestre el elemento. Los filtros se utilizan comúnmente para ajustar la representación de imágenes, fondos o bordes.

blur()

Permite realizar un desenfoque o difuminado, generalmente medido en pixels, a nuestros elementos HTML -Imágenes, texto, contenedores, entre otros.

filter: blur(6px);

Un posible uso consiste en activar el desenfoque al pasar el cursor por encima utilizando el selector :hover

          
            img:hover {
              filter: blur(6px);
            }

brightness()

Se utiliza para configurar el brillo de una imagen, toma como parámetro un número o porcentaje. Si el valor de este parámetro es 1 (o 100%) la imagen no cambiará. Si el valor es más pequeño que 1 la imagen se oscurecerá. El valor por defecto de del parámetro de brightness() es 0, y por tanto filter:brightness() produce una imagen completamente negra.

constrast()

Modifica el contraste de la imágen. Un valor de 0% mostrará la imagen gris y 100% muestra la imagen sin cambios. Acepta valores superiores a 100%, mostrando más contraste.

grayscale()

Nos permite mostrar la imagen en escala de grises. El parámetro define la proporción de la conversión. 100% mostrará completamente en escala de grises. 0% mostrará la imagen sin cambios. Los valores entre 0% y 100% son multiplicadores lineales del efecto. Por defecto el valor es 100%.

sepia()

Mostrará la imagen como sepia. El parámetro define la proporción de conversión. 100% es completamente sepia. 0% mostrará la imagen sin cambios. Los valores entre 0% y 100% son multiplicadores del efecto. Por defecto el parámetro es 100%.

hue-rotate()

Mostrará la imagen con una rotación de tono. El parámetro define el número de grados. 0 grados no mostrará cambios en la imagen, también es el valor por defecto. El valor máximo es 360 grados.

invert()

Mostrará la negativa de la imagen. El parámetro define la proporción. Un valor de 100% invierte completamente. Un valor de 0% no plaica cambios. El valor por defecto es 100%.

Si lo combinamos con grayscale, podemos conseguir el negativo de la imagen en blanco y negro.

          
            .filter {
              filter: grayscale(1) invert(1);
            }

opacity()

Podemos modificar la opacidad utilizando la propiedad opacity. Pero también podemos utilizar el filtro opacity. El parámetro define la proporcionalidad opacidad. El parámetro 1 o 100% mostrará la imagen completamente opaca. A valores más pequeños, la imagen se vuelve más transparente.

saturate()

Modifica la saturación de la imagen. El parámetro define en qué proporción se modificará. Hace lo mismo que la función grayscale(), pero al revés. 0% será completamente desaturado (escalade grises). 100% no cambiará la entrada. Pero también podemos aplicar valores más grandes que 1 o 100%. El resultado será una imagen de colores intensos, saturados.

Filtros a fondos (backdrop-filter)

Podemos aplicar flitros a los fondos, al área detrás de los elementos. Debido a que se aplica a todo lo que está detrás del elemento, para ver el efecto debe hacer que el elemento o su fondo sean al menos parcialmente transparentes.

De la misma manera que podemos aplicar filtros múltiples, también podemos apicarlos sobre el fondo.

backdrop-filter: blur(1rem) hue-rotate(240deg) opacity(0.75) invert(1);

Modos de Mezcla

Los modos de fusión se usan comúnmente en herramientas de diseño como Photoshop para crear un efecto de composición mezclando colores de dos o más capas.

mix-blend-mode

La propiedad mix-blend-mode define cómo el contenido de un elemento debe combinarse con su fondo.

  • initial: valor por defecto. No establece un modo de fusión.
  • inherit: heredará el modo de fusión de su padre
  • unset: elimina el modo de fusión actual
  • <blend-mode>: indica el modo de fusión
    • normal: no aplica mezcla alguna
    • multiply: es como apilar varias transparencias una encima de la otra. Los píxeles blancos aparecerán transparentes y los píxeles negros aparecerán negros. Los intermedios multiplicarán sus valores de luminosidad. Las luces se vuelven mucho más claras y las oscuras, más oscuras
    • screen: multiplica los valores de luz, un efecto inverso a la multiplicación y, en la mayoría de los casos, producirá un resultado más brillante
    • overlay: combina multiply y screen. Los colores oscuros se volverán más oscuros y los colores claros se volverán más claros. Los intermedios no se ven afectados.
    • darken: compara la luminosidad del color oscuro de la imagen de origen y la imagen de fondo y selecciona el más oscuro de los dos. Lo hace comparando los valores rgb en lugar de la luminosidad (como multiply y screen), para cada canal de color. Con darken y lighten, a menudo se crean nuevos valores de color a partir de este proceso de comparación
    • lighten: hace lo contrario que darken
    • color-dodge: aclara el color de fondo para reflejar el color de origen. Los colores negros puros no ven ningún efecto en este modo
    • color-burn: es muy similar al modo de fusión multiply, pero aumenta el contraste, da como resultado tonos medios más saturados y reflejos reducidos
    • hard-light: crea un contraste vívido. Filtra o multiplica los valores de luminosidad. Si el valor del píxel es más claro que un 50 % de gris, la imagen se aclara, como si estuviera tramada. Si es más oscuro, se multiplica
    • soft-light: es una versión menos dura de la superposición. Funciona de la misma manera con menos contraste.
    • difference: es como funciona un negativo fotográfico. Toma el valor de diferencia de cada píxel, invirtiendo los colores claros. Si los valores de color son idénticos, se vuelven negros. Las diferencias en los valores se invertirán
    • exclusion: es similar a difference, pero en lugar de devolver el negro para píxeles idénticos, devolverá un 50 % de gris, lo que dará como resultado una salida más suave con menos contraste
    • hue: toma el tono del color de origen y lo aplica a la saturación y luminosidad del color de fondo
    • saturation: funciona como el tono, pero al usar la saturación como modo de fusión, se aplica la saturación del color de origen al tono y la luminosidad del color de fondo
    • color: creará un color a partir del tono y la saturación del color de origen y la luminosidad del color de fondo
    • luminosity: es lo inverso del color. Crea un color con la luminosidad del color de origen y el matiz y la saturación del color de fondo

            </article>

              /* mix-blend-mode: normal;
              mix-blend-mode: color;

              mix-blend-mode: saturation;
              mix-blend-mode: luminosity;

              mix-blend-mode: soft-light;
              mix-blend-mode: lighten;
              mix-blend-mode: darken;
              mix-blend-mode: overlay; */

            }

background-blend-mode

La propiedad background-blend-mode describe cómo las imágenes de fondo y el color de fondo del elemento se deben mezclar entre sí.

Los valores que puede tomar la propiedad son las mismas que para la propiedad mix-blend-mode, pero en lugar de mix utiliza la palabra backgroun-blend-mode.

clip-path

La propiedad clip-path nos permite establecer un área específica de un elemento a mostrar. Crea una región de recorte que establece qué parte de un elemento debe mostrarse. Se muestran las partes que están dentro de la región, mientras que las que están fuera están ocultas.

La propiedad clip-path Utiliza 4 funciones (círcilo, elipse, rectangular y polígonos)

  1. clip-path: circle(); A través del parámetro podemos especificar un radio. El enmascaramiento comienza desde el centro y podemos utilizar valores como 100px at top (realizará el recorte del centro hacia arriba), at left, at right, at bottom, y podemos combinarlas como at left bottom. También podemos indicar coordenadas indicando donde realizar el recorte o a través de un porcentaje que aplicará en eje x y en eje y. Si se epecifican dos valores, el primero será para x y el segundo para y.
    2.             
              clip-path: circle();
              clip-path: circle(150px);
              clip-path: circle(150px at bottom right);
              clip-path: circle(150px at 0 0);
              clip-path: circle(150px at 30%);
              clip-path: circle(150px at 30% 20%);
  2. clip-path: ellipse(); Para elipses necesitamos indicar un radio para x y un radio para y. Aplican los mismo valores para posicionamiento del recorte.
    3.             
              clip-path: ellipse();
              clip-path: ellipse(100px 50px);
              clip-path: ellipse(100px 50px at top left);
              clip-path: ellipse(100px 50px at 0 0);
              clip-path: ellipse(100px 50px at 30%);
              clip-path: ellipse(100px 50px at 30% 20%);
  3. clip-path: inset(2rem); Requiere al menos un parámetro que funciona como si fuera padding. A mayor tamaño mayor será el recorte.

    4. Parámetros:

    • 1 valor, aplica el mismo padding a los 4 lados
    • 2 valores, el primero para arriba y abajo y el segundo para los lados
    • 3 valores, el primero para arriba, el segundo para los laterale y el tercero para abajo
    • 4 valores, indicamos los cuatro lados comenzando desde arriba y siguiendo las agujas del reloj

    También podemos redondear las esquinas de la máscara a través de la palabra clave round. Rpound acepta desde 1 a 4 parámetros.

    • 1 valor, aplicará el mismo redondeo a las 4 esquinas
    • 2 valores, el primer valor aplica a la diagonal (superior izquierda e inferior derecha) y el segundo para la otra diagonal
    • 3 valores, el primer valor es para la esquina superior izquierda, el segundo es para la diagonal (superior derecha e inferior izquierda) y el tercer valor es para la esquina inferior derecha
    • 4 valores, indicamos las 4 esquinas comenzando desde la esquina superior izquierda y siguiendo las agujas del reloj
                
              clip-path: inset(2rem);
              clip-path: inset(1rem 2rem);
              clip-path: inset(3rem 2rem 1rem);
              clip-path: inset(1rem 3rem 2rem 1rem);
              clip-path: inset(1rem round 1rem);
  4. clip-path: polygon(); A través de los parámetros definiremos los distintos nodos. A través de 3 modos podemos generar un recorte en forma de triángulo. Agregando un cuarto nodo formaremos un rectángulo. Podemos agregar tantos nodos como deseemos. Existen herramientas on-line como clippy para generar el código 
    5.             
              clip-path: polygon(0 0, 100% 0, 50% 100%);
              clip-path: polygon(50% 0%, 61% 35%, 98% 35%, 68% 57%, 79% 91%, 50% 70%, 21% 91%, 32% 57%, 2% 35%, 39% 35%);

            </article>

              /* clip-path: circle(150px at left);
              clip-path: circle(150px at 20% 50%);
              clip-path: ellipse();
              clip-path: inset(2rem);
              clip-path: inset(1rem round 1rem);

              clip-path: polygon(
                50% 0%,

shape-outside

Ejemplos de formas y siluetas entre textos e imágenes

Sabemos que el texto situado en una caja flotante se adapta para que fluya alrededor del elemento posicionado.
