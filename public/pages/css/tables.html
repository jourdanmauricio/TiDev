Tablas en CSS

Veremos algunas propiedades relacionadas con las tablas.

border-collapse;

Define si los bordes de la tabla se deben separar o colapsar.

  • border-collapse: separate; Cada celda de la tabla mostrará sus propios bordes. En este ejemplo, cada celda tiene un ancho de borde de 4px. Como resultado, el borde entre dos celdas será de 8 px.
  • border-collapse: collapse; Las celdas de tabla adyacentes fusionarán sus bordes. La celda que aparece primero en el código "ganará": sus bordes enmascararán los de las siguientes celdas.

empty-cells

La propiedad empty-cells selecciona celdas de tabla vacías con el fin de especificar si mostrar o no bordes y fondos en ellas. En otras palabras, le dice al navegador si debe o no dibujar bordes alrededor de una celda de la tabla o rellenar el fondo cuando esa celda no tiene contenido.

Esta es una propiedad interesante porque le da a CSS la capacidad de verificar el marcado HTML en busca de contenido dentro de una tabla y responder en consecuencia. Normalmente no pensamos en CSS como un lenguaje dinámico, pero esta es una instancia en la que se acerca bastante. Un buen caso de uso para celdas vacías podría ser una situación en la que no sepa si una tabla contendrá o no puntos de datos vacíos y decida ocultarlos.

  • empty-cells: show; mostrar un borde y un fondo en una celda vacía
  • empty-cells: hide; no muestra un borde o fondo en una celda vacía
  • empty-cells: inherit; hereda el valor de la propiedad del elemento padre
  • empty-cells: initial; usar el valor predeterminado definido para la propiedad
  • empty-cells: unset; resetea la propiedad a su valor heredado

border-spacing

Es la distancia entre los bordes de celdas adyacentes (sólo para el modelo de separación de borde). Ésta propiedad sólo se aplica cuando border-collapse es especificado en separate.

El espaciado de borde también se utiliza en los bordes exteriores de las tablas donde la distancia entre los bordes de tabla y las celdas en la primera/última columna o fila es la suma de los espaciados (horizontales o verticales) y de los espaciados (arriba, derecho, abajo o izquierdo) en una tabla.

        
          border-collapse: separate;
          border-spacing: 10px 5px;
        
      
        
          <table class="table">
            <tr>
              <th>A</th>
              <th>B</th>
              <th>C</th>
            </tr>
            <tr>
              <td>1</td>
              <td>2</td>
              <td>3</td>
            </tr>
            <tr>
              <td>4</td>
              <td>5</td>
              <td>6</td>
            </tr>
            <tr>
              <td>7</td>
              <td></td>
              <td>9</td>
            </tr>
          </table>  
        
      
        
          .table {
            font-size: 2rem;
            border-collapse: collapse;
            border-collapse: separate; /* Default */
            empty-cells: show; /* Default */
            empty-cells: hide;
          }
          .table,
          th,
          td {
            border: medium solid teal;
            border-spacing: 1rem;
          }
