Estilos de Imágenes

Imágenes resṕonsivas

Una de las principales características de las imágenes es que pueden adaptarse al tamaño de la pantalla. Si la imagen es más grande que el contenedor se desbordará.

Si deseamos que la imagen sea recursiva tenemos que setear la propidad de las imagenes max-width en 100%.

La etiqueta img es una etiqueta con comportamiento en línea. Si la imagen es menor al tamaño del contenedor otros elementos se colocarán a su lado.

          
            <img src="/assets/images/puesta-sol.jpg" alt="La puesta del sol">
          
        
          
            .img {
              max-width: 100%;
              height: auto;
            }

Imágenes con tamaño específico

Si necesitamos que un imagen tenga dimensiones específicas podemos incluirla dentro de un contenedor y dar dimensiones al contendor.

          
            <article class="card">
              <img class="image-image" src="/assets/images/puesta-sol.jpg" alt="La puesta del sol">
              <h4>Puesta del sol</h4>
            </article>
          
        
          
            .card {
              border: medium solid black;
              width: 400px;
              height: 400px;
            }
            .card img {
              height: 300px;
            }

Los problemas empiezan cuando queremos que la imagen ocupe un alto particular. Si las dimensiones de la imagen con respecto al contendor no son proporcionales se deformará. Para solucionar este problema existen dos propiedades: object-fit y object-position.

La propiedad object-fit define cómo responde un elemento a la altura y el ancho de su contenedor. Está diseñado para imágenes, videos y otros formatos de medios junto con la propiedad object-position. Usado solo, object-fit nos permite recortar una imagen en línea brindándonos un control detallado sobre cómo se aplasta y se estira dentro de su caja.

object-fit admite cinco valores:

  • fill: este es el valor predeterminado. Estira la imagen para que se ajuste al contenedor, independientemente de su aspect-ratio
  • contain: aumenta o disminuye el tamaño de la imagen para llenar el cuadro conservando su aspect-ratio
  • cover: la imagen llenará la altura y el ancho del contendor, una vez más manteniendo su relación de aspecto, pero a menudo recortando la imagen en el proceso
  • none: la imagen ignorará el alto y el ancho del padre y conservará su tamaño original
  • scale-down: la imagen comparará la diferencia entre none y contain para encontrar el tamaño de objeto concreto más pequeño

La propiedad object-position determina el alineamiento del elemento dentro de la caja.

          
            <article class="card2">
              <img src="/assets/images/puesta-sol.jpg" alt="La puesta del sol">
              <h4>Puesta del sol</h4>
            </article>
          
        
          
            .card2 {
              border: medium solid black;
              width: 400px;
              height: 400px;
            }
            .card2 img {
              height: 300px;
              background-color: aquamarine;
              /* la imagen se deforma */
              object-fit: fill;
              /* cover no deforma la imagen pero se recorta centrando la imagen */
              object-fit: cover;
              /* porque activa la propiedad object-position que por defecto tiene el valor: 50% 50% */
              /* hacemos que desborde por derecha y abajo, ya que seteamos inicio arriba e izquierda */
              object-position: left top;
              /* el valor contain ajusta la imagen y no la deforma, pero no respeta la altura definida en height */
              object-fit: contain;
              /* Respetamos las dimensiones de la imagen. Se recorta. */
              object-fit: none;
              object-position: -40px 50%;
              /* Si agregamos object-position: 50% 50%; centramos la imagen */
              object-position: 50% 50%;
              /* scale-down toma el menor valor entre none y contain */
              /* si none es menos que contain toma none; si contain es menor que none toma contain */
              object-fit: scale-down;
            }
