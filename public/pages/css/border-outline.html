Bordes y outlines

Redondeo de bordes

La propiedad border-radius nos permite redondear los bordes de una caja. Define el radio del elemento. Border-radius también es un shorthand.

          
            <div class="box border-radius-1"></div>
          
        
          
            .box {
              background-color: black;
              width: 400px;
              height: 400px;
              margin-left: auto;
              margin-right: auto;
              border: thick solid red;
            }
            .border-radius-1 {
              /* Apicando 4 valores */
              border-top-left-radius: 4rem;
              border-top-right-radius: 4rem;
              border-bottom-right-radius: 4rem;
              border-bottom-right-radius: 4rem;
              /* Aplicando 2 valores */
              border-radius: 2rem 4rem; /* (sup izq - inf der / sup der - inf izq) */
              /* Aplicando 3 valores */
              border-radius: 2rem 4rem 0rem; /* (sup izq / sup der - inf izq / inf der) */
              /* Aplicando 4 valores */
              border-radius: 4rem 3rem 2rem 1rem; /* Empieza sup izq y sigue las agujas del reloj */
              border-radius: 2rem; /* Shorthand */
            }

También podemos aplicar bordes elípticos dividiendo cada valor en dos valores, uno para el radio en el eje x y otro para el eje y. Si detallamos por esquina, se utiliza el separador / colocando a la izquierda los valores para x y a la derecha de la / los valores para y.

          
            <div class="box border-radius-2"></div>
          
        
          
            .border-radius-2 {
              border-radius: 2rem / 4rem;
              border-radius: 2rem 4rem 0 / 4rem 8rem 0;
            }

A través de la propiedad border-radius también podemos redondear tanto los border hasta llegar a un círculo. Se utiliza mucho para las fotos de perfil en cards. Si la caja a redondear no es un cuadrado exacto se formará una elipsis. 

          
            <div class="box border-radius-3"></div>
          
        
          
            .border-radius-3 {
              border-radius: 50%;
            }

Outlines

Outline es un borde exterior. No existe dentro del modelo de caja. Sus propiedades son outline-width, outline-style, outline-color y outline-offset.

No forma parte de la caja. En cambio, el borde si forma parte de la caja. El outline es un borde externo que no forma parte dela caja.

La propiedad outline-offset lo separa de la caja y acepta valores positivos o negativos.

La porpiedad outline es utilizada por los elementos html de los formularios trabajando sobre el evento focus. 

          
            <div class="box outline"></div>
          
        
          
            .outline {
              outline-width: 5px;
              outline-style: dotted;
              outline-color: aquamarine;
              outline: thick solid yellowgreen;
              color: white;
              text-align: center;
              outline-offset: -2rem;
            }
