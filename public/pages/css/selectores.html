Selectores CSS

Los selectores se utilizan para seleccionar los elementos HTML de nuestra página a los que queremos aplicar estilo. Existen muchos selectores CSS que permiten una gran precisión a la hora de seleccionar elementos a los que aplicar estilo.

Selectores básicos: Etiquetas, Identificadores y Clases

Existen 3 selectores básicos

  • Etiquetas arrow_right_alt se asocian con elementos HTML. Cuando aplicamos estilos a una etiqueta impactará sobre todas las etiquetas de ese tipo del documento. 
                
              <p>Lorem ipsum dolor sit amet consectetur, adipisicing elit. Tenetur enim quas ut officia quidem minus 
              sapiente quibusdam consequatur quaerat facere.<p>
            
          
                
              p {
                color: blue;
              }

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Asperiores qui harum eius deleniti neque quaerat ad corporis nostrum facilis, error vitae cupiditate quasi omnis laborum natus cum sequi id molestias?

  • Identificadores arrow_right_alt en los elementos HTML debemos agregar el atributo id y en css lo indentificamos por el símbolo numeral (#). En este caso el estilo solo se aplicará sobre el elemento HTML identificado por ese id. Es una mala práctica utilizar los id para establecer los estilos. Dentro de los documentos, los identificadores son únicos por lo que no estaríamos reutilizando los estilos en varios elementos. Por otro lado, más adelante veremos el algoritmo de CSS y cómo se trata la especificidad de selectores. Allí veremos porqué es un anti patrón utilizar los id para los estilos. Los id se utilizan más en relación a JavaScript. 
                
              <p id="parrafo-dos">Lorem ipsum dolor sit amet consectetur, adipisicing elit. Tenetur enim quas ut 
              officia quidem minus sapiente quibusdam consequatur quaerat facere.<p>
            
          
                
              #parrafo-dos {
                background-color: pink;
              }

    Lorem ipsum dolor sit amet consectetur, adipisicing elit. Tenetur enim quas ut officia quidem minus sapiente quibusdam consequatur quaerat facere.

  • Clases arrow_right_alt en los elementos HTML debemos agregar el atributo class y en css lo indentificamos por el símbolo del punto (.). Las clases nos permiten reutilizar código, se aplicarán a todos los elementos que posean la clase asignada. Una de las ventajas es que en el documento HTML podemos agregar más de una clase a un elemento. Solo se separan las clases que le queremos asignar por un espacio.

    Como las clases son reutilizables tenemos que darles nombres alusivos a las acciones que realizan.

                
              <p class="text-32 bg-skyblue">Lorem ipsum dolor sit amet consectetur, adipisicing elit. 
              Tenetur enim quas ut officia quidem minus sapiente quibusdam consequatur quaerat facere.<p>
            
          
                
              .text-32 {
                font-size: 32px;
              }

              .bg-skyblue {
                background-color: skyblue;
              }

    Lorem ipsum dolor sit amet consectetur, adipisicing elit. Tenetur enim quas ut officia quidem minus sapiente quibusdam consequatur quaerat facere.

Selectores avanzados

Selectores de hijos

Existen dos tipos de selectores de tipo hijos.

Selector de hijos directos

Un hijo directo es un elemento que se encuentra en el primer nivel del elemento referenciado con la clase. En el siguiente ejemplo las etiquetas li son hijas directas del ul pero las etiquetas b, a pesar de ser hijas, no son hijas directas. Son "nietas". En CSS la nomenclatura para indicar hijo directo es el símbolo mayor (>). 

            
              <ul class="hijos-directos">
                <li><b>Primavera</b></li>
                <li><b>Verano</b></li>
                <li><b>Otoño</b></li>
                <li><b>Invierno</b></li>
              </ul>
            
          
            
              .hijos-directos > li {
                background-color: thistle;
              }
  
              /* no aplica porque b no es hijo directo de ul */
              .hijos-directos > b {
                background-color: thistle;
              }
  • Primavera
  • Verano
  • Otoño
  • Invierno

Selector de hijos descendientes

En CSS, el selector de hijo descendiente se representa por un espacio en blanco y aplica a todos los hijos del elemento si considerar el orden en el que se encuentren. No importa si son directos o no.

            
              <ul class="hijos-descendientes">
                <li><b>Primavera</b></li>
                <li><b>Verano</b></li>
                <li><b>Otoño</b></li>
                <li><b>Invierno</b></li>
              </ul>
            
          
            
              .hijos-descendientes b {
                background-color: thistle;
              }
  • Primavera
  • Verano
  • Otoño
  • Invierno
El fondo se pinta distinto porque en el primer caso aplica sobre los li (elementos de bloque), mientras que en el segundo caso aplica sobre las b (elementos en línea).

Selectores de hermanos

Existen dos tipos de selectores de tipo hijos.

Los selectores de hermanos aplicarán sobre los elementos hermanos (mismo nivel) pero que se encuentren por debajo del elemento de referencia.

Selector de hermanos (en general)

El selector de hermanos en general se representa por el símbolo virgulilla (~) y aplicará a los elementos espcificados que se encuentren en el mismo nivel. En el siguiente ejemplo aplicará sobre todos los li que se encuentren en el mismo nivel que el elemento que posee la clase que referenciamos.

            
              <ul>
                <li><b>Primavera</b></li>
                <li class="hermanos-general"><b>Verano</b></li>
                <li><b>Otoño</b></li>
                <li><b>Invierno</b></li>
              </ul>
            
          
            
              .hermanos-general ~ li {
                background-color: thistle;
              }
  • Primavera
  • Verano
  • Otoño
  • Invierno

¿Por qué no aplica al elemento li Primavera? Porque los selectores de hermanos aplican a elementos que sean hermanos pero que se encuentren por debajo de él. No aplican hacia arriba.

Selector hermano adyacente

El selector hermano adyacente se representa por el símbolo suma (+) y aplicará al siguiente elemento que se encuentren en el mismo nivel del elemento referenciado. En el siguiente ejemplo aplicará sobre el li Otoño.

            
              <ul>
                <li><b>Primavera</b></li>
                <li class="hermanos-adyacentes"><b>Verano</b></li>
                <li><b>Otoño</b></li>
                <li><b>Invierno</b></li>
              </ul>
            
          
            
              .hermanos-adyacentes + li {
                background-color: thistle;
              }
  • Primavera
  • Verano
  • Otoño
  • Invierno

Selectores de atributos

Nos permite aplicar selectores a través de cualquier atributo relacionado a etiquetas HTML. Se representan a través de los corechetes [].

            
              <nav class="seletores-atributos">
                <ul>
                  <li><a href="#" target="_blank">Inicio</a></li>
                  <li><a href="#" target="_blank">Cursos</a></li>
                  <li><a href="#" target="_blank">Blog</a></li>
                  <li><a href="#" target="_blank">CV</a></li>
                  <li><a href="#" target="_blank">Ahora</a></li>
                  <li><a href="#" target="_blank">Notas</a></li>
                </ul>
              </nav>
            
          
            
              .seletores-atributos a[href] {
                color: orangered;
              }
  • Inicio
  • Cursos
  • Blog
  • CV
  • Ahora
  • Notas

Aplica a todos los elementos a dentro del elemento que contenga la clase "selectores-atributos" y que posean el atributo "href". También podemos agregar el valor que posee el atributo para realizar la selección. Ejemplo:

            
              .seletores-atributos a[target="_blank"] {
                font-weight: bold;
              }

Otra característica de los selectores de atributos es que pueden utilizar comodines

  • * arrow_right_alt el valor del atributo contiene los caracteres que se detallen
  • ^ arrow_right_alt el valor del atributo inicia con los caracteres que se detallen
  • $ arrow_right_alt el valor del atributo finaliza con los caracteres que se detallen
  • ~ arrow_right_alt el valor del atributo contiene el valor y se encuentra en una lista separada por espacios
  • | arrow_right_alt el valor del atributo contiene el valor y se encuentra en una lista separada por espacios y que después del texto contenga un guión medio
            
              .seletores-atributos a[target*="bl"] {
                font-weight: bold;
              }
              .seletores-atributos a[target^="_bl"] {
                font-weight: bold;
              }
              .seletores-atributos a[target$="anc"] {
                font-weight: bold;
              }
              .seletores-atributos a[class~="boton"] {
                font-weight: bold;
              }
              .seletores-atributos a[class|="menu"] {
                font-weight: bold;
              }

Selector universal

Se representa por el asteristo (*) y aplica a todos los elementos HTML que existan en el documento.

          
            * {
              font-family: sans-serif;
            }

Aplica la tipografía sans-serif a todos los elementos, y al ser una tipografía estandar existe en todos los sistemas operativos (Mac, Windows o Linux).

También se puede aplicar el selector universal pero contextualmente. Aplicará a todos los elementos que se encuentren dentro del elemento referenciado.

          
            <ul class="selector">
              <li><b>Primavera</b></li>
              <li><b>Verano</b></li>
              <li><b>Otoño</b></li>
              <li><b>Invierno</b></li>
            </ul>
          
        
          
            .selector * {
              background-color: thistle;
            }

Pseudoclases

Las pseudoclases aplicarán los estilos dependiendo del contexto, de la posición del elemento o de ciertos estados que tenga elemento HTML. Las pseudoclases se representan a través de los dos puntos (:).

Documentación de las distintas pseudoclases que podemos encontrar: Pseudoclases

Pseudoclases para elementos de formularios

  • required arrow_right_alt representa cualquier elemento input, select o textarea que tenga el atributo requerido establecido.
  • valid arrow_right_alt representa cualquier elemento input u otro del form cuyo contenido se valide correctamente. Esto permite hacer que los campos válidos adopten una apariencia que ayude al usuario a confirmar que su dato se ingresó correctamente.
  • invalid arrow_right_alt representa cualquier fieldset, input u otro elemento del form cuyo contenido no se valide correctamente. Esta pseudoclase es útil para resaltar errores para el usuario.
  • focus arrow_right_alt representa un elemento (como una entrada de formulario) que ha recibido el foco. Por lo general, se activa cuando el usuario hace clic o toca un elemento o lo selecciona con la tecla tabulador del teclado.
  • disabled arrow_right_alt representa cualquier elemento deshabilitado. Un elemento está deshabilitado si no se puede activar (seleccionar, hacer clic, escribir, etc.) o aceptar el enfoque.
            
              <form class="form-pseudoclase" action="#">
                <input type="text" name="nombre" placeholder="Escribe tu nombre" required>
                <br>
                <input type="email" name="correo" placeholder="Escribe tu correo" required>
                <br>
                <input type="submit">
              </form>
            
          
            
              form-pseudoclase *:required {
                border-color: yellow;
              }
              form-pseudoclase *:valid {
                border-color: green;
              }
              form-pseudoclase *:invalid {
                border-color: red;
              }
              form-pseudoclase *:focus {
                background-color: skyblue;
              }
              form-pseudoclase *:disabled {
                background-color: pink;
              }



Pseudoclases para elementos de enlace

  • hover arrow_right_alt aplica sobre cualquier elemento HTML. Se activa cuando el usuario pasa el cursor sobre un elemento (puntero del mouse). Los estilos definidos por la pseudoclase :hover serán anulados por cualquier pseudoclase posterior relacionada con el enlace (:enlace, :visitado o :activo) que tenga al menos la misma especificidad. Para aplicar un estilo adecuado a los enlaces, coloque la regla :hover después de las reglas :link y :visited, pero antes de :active
  • visited arrow_right_alt enlaces que el usuario ya ha visitado
  • active arrow_right_alt representa un elemento (como un botón) que está siendo activado por el usuario. Cuando se usa un mouse, la "activación" generalmente comienza cuando el usuario presiona el botón principal del mouse
  • link arrow_right_alt representa un elemento que aún no ha sido visitado. Coincide con todos los elementos a no visitados que tienen un atributo href
  • target arrow_right_alt representa un elemento único (el elemento objetivo) con una identificación que coincide con el fragmento de la URL. Cuando la url actual posee el símbolo numeral (#) y se establece la pseudoclase :target se aplicarán los estilos a ese identificador 
                    
                  /* Selecciona un elemento con un ID que coincida con el fragmento de la URL actual */
                  :target {
                    border: 2px solid black;
                  }
De acuerdo a la especificidad de las pseudoclases, deberían definirse en con el siguiente orden:
  1. link
  2. active
  3. visited
  4. hover
            
              <nav class="menu-pseudoclases">
                <a href="#seccion1">Seccion 1</a>
                <a href="#seccion2">Seccion 2</a>
                <a href="#seccion3">Seccion 3</a>
                <a href="#seccion4">Seccion 4</a>
              </nav>
            
          
            
              .menu-pseudoclases a:link{
                color: pink;
              }
              .menu-pseudoclases a:active{
                color: red;
              }
              .menu-pseudoclases a:visited{
                color: olive;
              }
              .menu-pseudoclases a:hover{
                color: orange;
              }

Pseudoclases por posición y tipo

Las pseudoclases por posición hacen referencia a los elementos en relación a la posición que ocupan dentro de su contenedor.

  • first-child arrow_right_alt aplicará al primer elemento que tenga que sea del tipo indicado
  • last-child arrow_right_alt aplicará al último elemento que tenga que sea del tipo indicado
  • nth-child(nro) arrow_right_alt aplicará al elemento nro que tenga que sea del tipo indicado. Esta pseudoclase admite determinadas fórmulas
    • nth-child(4n) arrow_right_alt aplicará al elemento que tenga que sea del tipo indicado pero cada 4 elementos. Aplicará al 4, 8, 12, etc
    • nth-child(3n+1) arrow_right_alt aplicará al elemento que tenga que sea del tipo indicado pero cada 3 elementos + 1. Aplicará al 4, 7, 10, etc
    • nth-child(even) arrow_right_alt aplicará a los elementos pares que sean del tipo indicado
    • nth-child(odd) arrow_right_alt aplicará a los elementos impares que sean del tipo indicado
            
              <ol class="lista-pseudoclases">
                <li>La Beriso</li>
                <li>Bersuit Vergarabat</li>
                <li>Las Blacanblus</li>
                <li>Callejeros</li>
                <li>Hermética</li>
                <li>Divididos</li>
                <li>La Mississippi</li>
                <li>Las Pelotas</li>
                <li>Pappo's Blues</li>
                <li>La Renga</li>
                <li>Los Piojos</li>
                <li>Catupecu Machu</li>
              </ol>
            
          
            
              .lista-pseudoclases li:first-child {
                color: green;
              }
              .lista-pseudoclases li:last-child {
                color: violet;
              }
              .lista-pseudoclases li:nth-child(3) {
                color: navy;
              }
              .lista-pseudoclases li:nth-child(4n) {
                font-size: 20px;
              }
              .lista-pseudoclases li:nth-child(3n+1) {
                font-weight: bold;
              }
              .lista-pseudoclases li:nth-child(even) {
                background-color: yellow;
              }
              .lista-pseudoclases li:nth-child(odd) {
                background-color: orange;
              }

Pseudoclases por posición y tipo

Este tipo de pseudoclases no solo aplican por posición sino que también consideran el tipo de etiqueta.

  • first-of-type arrow_right_alt buscará dentro de la clase el primer tipo que coincida con el especificado
  • last-of-type arrow_right_alt buscará dentro de la clase el último tipo que coincida con el especificado
  • nth-of-type(nro) arrow_right_alt buscará dentro de la clase el número de elemento del tipo indicado por parámetro que coincida con el especificado. También aplican las fórmulas nth-of-type(4n), nth-of-type(3n+1), nth-of-type(even) y nth-of-type(odd)
            
              <article class="articulo-pseudoclases">
                <header>Este es un header</header>
                <p>Este es un párrafo</p>
                <div>Este es un div</div>
                <p>Este es otro párrafo</p>
                <section>Esta es una sección</section>
                <p>Este es otro párrafo</p>
                <div>Este es un div</div>
                <p>Este es otro párrafo</p>
                <footer>Este es un footer</footer>
              </article>
          
          
            
              .articulo-pseudoclases p:first-of-type {
                background-color: pink;
              }
              .articulo-pseudoclases p:last-of-type {
                background-color: violet;
              }
              .articulo-pseudoclases p:nth-of-type(3) {
                background-color: fuchsia;
              }

Pseudoclases de negación

Podemos utilizar esta pseudoclase cuando queremos aplicar a todos los elementos con excepción de un tipo. Si necesitamos que sean más de un elemento a exluir podemos separarlos por coma.

            
              .articulo-pseudoclases *:not(p, boton, .clase) {
                background-color: beige;
              }

Pseudoelementos

Las pseudoelementos aplicarán los estilos a partes específicas de un elemento y se representan a través de los dos dos puntos (::).

Documentación de los distintos pseudoelememtos que podemos encontrar: Pseudoelememtos

  • after y before arrow_right_alt nos permiten agregar contenido antes o después de un elemento en particular
  • first-letter arrow_right_alt los estilos se aplicarán al primer caracter
  • first-line arrow_right_alt los estilos se aplicarán a la primera línea
  • selection arrow_right_alt los estilos se aplicarán al momento de seleccionar el elemento
  • placeholder arrow_right_alt los estilos se aplicarán al atributo placeholder del elemento input refernciado
          
            <i>Bandai Myth Cloth Ex</i> <span class="saga-geminis">Saga geminis</span> <span class="saga-precio">500</span>
            <p class="p-seudoelementos">

              perferendis libero optio, esse repudiandae minima sequi voluptatum provident magnam excepturi aperiam temporibus
              exercitationem. Quos!
              </p>

            <input class="input-pseudoelementos" type="text" name="nombre" placeholder="Escribe tu nombre">

Agrupación de selectores

Si necesitamos aplicar los mismos estilos a un conjunto de elementos podemos agruparlos separandolos por coma. 

          
            <form class="form-agrupar-selectores" action="#">
              <input type="text" name="nombre" placeholder="Escribe tu nombre">
              <input type="email" name="correo" placeholder="Escribe tu correo">
              <textarea name="comentarios" cols="30" rows="10" placeholder="Escribe tus comentarios"></textarea>
              <input type="submit" value="Enviar">
            </form>
          
        
          
            .form-agrupar-selectores input[type="text"],
            .form-agrupar-selectores input[type="email"],
            .form-agrupar-selectores textarea {
              display: block;
              border-color: yellow;
              border-width: 2px;
              background-color: black;
              font-size: 20px;
              width: 300px;
            }
