Estilos de Listas y Texto en columnas

Estilos en Listas

Las propiedades que podemos utilizar para asignar estilos a las listas son: list-style-images, list-style-position, list-style-type y list-style.

list-style es la propiedad abreviada para list-style-images, list-style-position y list-style-type.

list-style-type

Define el tipo de viñeta de un elemento de lista. También se puede realizar utilizando el atributo type del elemento ul.

Existen muchos tipos de viñetas, veremos los más comunes. Podemos ver el listado completo en lista de tipos.

  • list-style-type: disc; Los elementos de la lista utilizarán un disco como viñeta.
  • list-style-type: circle; Los elementos de la lista usarán un círculo como viñeta.
  • list-style-type: square; Los elementos de la lista usarán un cuadrado como viñeta.
  • list-style-type: decimal; Los elementos de la lista usarán un decimal como viñeta.
  • list-style-type: none; Los elementos de la lista no tendrán viñetas.
          
            <ul class="list">
              <li>Primavera</li>
              <li>Verano</li>
              <li>Otoño</li>
              <li>Invierno</li>
            </ul>
          
        
          
            .list {
              list-style-type: square;
            }
  • Primavera
  • Verano
  • Otoño
  • Invierno

list-style-images

Define la imagen que se utilizará como viñeta de un elemento de lista.

  • list-style-image: none; Los elementos de la lista utilizarán la viñeta definida por el valor de tipo de estilo de lista, que de forma predeterminada es un disco
  • list-style-image: url(/assets/images/list-style-image.png); Los elementos de la lista usarán la imagen ubicada en la URL especificada como su viñeta. La imagen no se puede cambiar de tamaño

list-style-position

  • list-style-position: outside; La viñeta estará fuera del elemento de la lista, como si no fuera parte del elemento de la lista. El inicio de cada línea de un elemento de la lista se alineará verticalmente
  • list-style-position: inside; La viñeta estará dentro del elemento de la lista. Como es parte del elemento de la lista, será parte del texto y empujará el texto al principio.
    •           
            <ul class="list-2">
              <li>Primavera</li>
              <li>Verano</li>
              <li>Otoño</li>
              <li>Invierno</li>
            </ul>
          
        
              
            .list-2 {
              list-style-position: outside;
              list-style-position: inside;
            }
            .list-2 li {
              background-color: coral;
            }
    • Primavera
    • Verano
    • Otoño
    • Invierno

    Texto en columnas

    Las siguientes propiedades no sirven para maquetar una página. Para maquetar se utiliza Grid y Flexbox. Esta técnica solo nos sirve para poner texto en columnas.

    Columnas para párrafos y listas

    Propiedades: column-count, column-rule, column-width y column-gap.

               
            <ol class="text-column-4">
              <li>La Beriso</li>
              <li>Bersuit Vergarabat</li>
              <li>Las Blacanblus</li>
              <li>Callejeros</li>
              <li>Hermética</li>
              <li>Divididos</li>
              <li>La Mississippi</li>
              <li>Las Pelotas</li>
              <li>Pappo's Blues</li>
              <li>La Renga</li>
              <li>Los Piojos</li>
              <li>Catupecu Machu</li>
            </ol>     
          
        
              
            .text-column-4 {
              column-count: 4;
              column-gap: 2rem;
            }
