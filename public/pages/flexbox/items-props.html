Propiedades para los items

order

De forma predeterminada, los elementos se presentan en el orden de origen. Sin embargo, la propiedad order controla el orden en que aparecen en el contenedor.

Los items con el mismo orden vuelven al orden de origen.

        
          .item {
            order: 5; /* default is 0 */
          }

flex-grow

Define la capacidad de crecimiento de un elemento. Acepta un valor sin unidades que sirve como proporción. Determina qué cantidad de espacio disponible dentro del contenedor debe ocupar el item.

Si todos los elementos tienen el crecimiento establecido en 1, el espacio restante en el contenedor se distribuirá por igual a todos los hijos. Si uno de los hijos tiene un valor de 2, ese hijo ocuparía el doble de espacio que cualquiera de los otros.

El valor por defecto es 0, no crezcas.

El factor de crecimiento no depende del espacio total, depende del espacio sobrante. Refiere a cómo se distribuye el espacio sobrante entre los elementos.

        
          .item {
            flex-grow: 4; /* default 0 */
          }

flex-shrink

Define la capacidad de que un elemento se encoja.

El valor por defecto es 1. Reducir tamaño.

Si todos los elementos tienen flex-shrink establecido en 1 y uno de los hijos tiene un valor de 2, ese hijo se reducirá el doble de espacio que cualquiera de los otros.

        
          .item {
            flex-shrink: 3; /* default 1 */
          }

flex-basis

Define el tamaño predeterminado de un elemento antes de que se distribuya el espacio restante. Puede ser una longitud (por ejemplo, 20 %, 5 rem, etc.) o una palabra clave. La palabra clave auto significa "mirar mi propiedad de ancho o alto" (que se hizo temporalmente por la palabra clave de tamaño principal hasta que quedó obsoleta). La palabra clave content significa "dimensionarlo según el contenido del elemento": esta palabra clave aún no es compatible, por lo que es difícil de probar y más difícil saber qué hacen sus hermanos contenido máximo, contenido mínimo y contenido adecuado.

Es el tamaño del elemento hijo dentro de la fila del contenedor. Si la dirección del contenedor es row, flex-basis representa el width. Si la dirección del contenedor es column, flex-basis representa el height.

El valor por defecto es auo.

        
          .item {
            flex-basis: auto | 100px; /* default auto */
          }

Si se establece en 0, el espacio adicional alrededor del contenido no se tiene en cuenta. Si se establece en automático, el espacio adicional se distribuye en función de su valor de crecimiento flexible.

flex

Es la abreviatura de flex-grow, flex-shrink y flex-basis combinados. Los parámetros segundo y tercero (flex-shrink y flex-basis) son opcionales. El valor predeterminado es 0 1 auto, pero si lo configura con un solo valor numérico, como flex: 5; es como configurar flex-grow: 5, flex-shrink: 1 y flex-basis: 0%. 

        
          .item {
            flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]
            flex: 0 1 auto /* Default. No crecer, reducir proporcionalmente, dimensión de contenido */
          }

align-self

Permite anular la alineación predeterminada para elementos individuales. Sobreescribe el valor que tenga definido el padre en la propiedad aling-items.

        
          .item {
            align-self: auto | flex-start | flex-end | center | baseline | stretch;
          }
