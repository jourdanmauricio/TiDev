Multimedia Flexible

Los elementos multimedia es todo aquello que no sea texto:

  • img
  • audio
  • video
  • iframe
  • canvas
  • svg
  • picture
  • object
  • embed
  • map
  • area

Los contenidos multemedia deben ser flexibles y adaptarse al tamaño de su contendor. Lo logramos con la siguiente regla:

         
          img, audio, video, iframe, canvas, svg, picture {
            max-width: 100%;
            height: auto;
          }

Imágenes responsivas

Adaptamos la imagen al tamaño del contenedor. Al asignar la propiedad max-width al 100% no desbordadrá en su ancho, y al asignar auto en height logramos que la imagen no se deforme.

        
          <img src="/assets/images/puesta-sol.jpg" alt="Puesta del sol">
        
      
        
          img {
            max-width: 100%;
            height: auto;
          }

Etiqueta srcset y sizes

Estos atributos mejoran el rendimiento de la carga al colocar la misma imágen pero en diferentes tamaños y pesos dependiendo de la resolución de pantalla. En el ejemplo, se aplicaron filtros a los distintos tamaños para identificarlos visualmente, pero tendríamos que utilizar la misma imagen con distintos tamaños.

No es necesario incrustar imágenes grandes en la página si se está viendo en una pantalla móvil. Y, a la inversa, una imagen pequeña comienza a verse más pixelada cuando se muestra mayor que su tamaño original.

Algunos dispositivos tienen pantallas de alta resolución que necesitan imágenes más grandes de las que se espera para que luzcan bien.

Las imágenes vectoriales resolverían estos problemas, y lo hacen hasta cierto punto: son pequeñas en tamaño de archivo y escalan bien, y deberías usarlos siempre que sea posible.

Utilizando la resolución de pantalla (aspect ratio):

        
          <img src="/assets/images/html5-3d-600.png"
          srcset="/assets/images/html5-3d-600.png 1x, /assets/images/html5-3d-900.png 1.5x, /assets/images/html5-3d-1200.png 2x"
          alt="HTML5">

Utilizando el ancho de la pantalla:

        
          <img src="/assets/images/html5-3d-600.png"
          srcset="/assets/images/html5-3d-600.png 600w, /assets/images/html5-3d-900.png 900w, /assets/images/html5-3d-1200.png 1200w"
          alt="HTML5">

El atributo size nos permite trabajar con media queries dentro de las imágenes. En el siguiente ejemplo, cuando el tamaño de pantalla sea menor a 480px la imagen se renderizará en 300px, entre 480px y 600px se mostrará en 480px, y así sucesivamente 

        
          <img src="/assets/images/html5-3d-600.png"
          srcset="/assets/images/html5-3d-600.png 600w, /assets/images/html5-3d-900.png 900w, /assets/images/html5-3d-1200.png 1200w"
          sizes="(max-width:480px) 300px, (max-width:600px) 480px, (max-width:768px) 600px, (max-width:1024px) 900px, (max-width:1200) 1200px"
          alt="HTML5">

Mientras el atributo srcset nos permite cambiar la imágen a la misma de menor tamaño (mejora de rendimiento), el atributo sizes nos permite redimencionar un tamaño específico.

Etiqueta picture

El objetivo de esta etiqueta, no es solo mostrar diferentes tamaños de la misma imagen, sino también mostrar imágenes diferentes. Por ejemplo, recortar una imagen de forma diferente según el tamaño de la pantalla y las diferencias en el diseño. Esto se conoce como "dirección de arte".

Un ejemplo es el logotipo, en dispositivos pequeños se utiliza una versión minificada del logo, y en desktop se visualiza el logo con el nombre de la empresa.

        
          <picture>
            <source srcset="/assets/images/kitten-large.png" media="(min-width:1024px)">
            <source srcset="/assets/images/kitten-medium.png" media="(min-width:600px)">
            <img src="/assets/images/kitten-small.png" alt="Kitten">
          </picture>

Con la etiqueta picture también podemos aplicar otro tipo de cambios responsivos. Ejemplo: imágenes con formato webp que no son soportadas por algunos navegadores.

        
          <picture>
          <source srcset="/assets/images/butterfly.webp" type="image/webp">
          <img src="/assets/images/butterfly.jpg" alt="Mariposa">
          </picture>

Videos Responsivos

        
          <video src="/assets/media/sunset_evening_deck_877.mp4" controls></video>

El video del ejemplo anterior posee un formato widescreen (16:9), pero si tuviera otro formato podemos hacerlo responsivo a través de una técnica que ya posee cierta antigüedad:

        
          <div class="responsive-media">
            <video src="/assets/media/sunset_evening_deck_877.mp4" controls></video>
          </div>
        
      
        
          .responsive-media {
            position: relative;
            width: 100%;
            height: 0;
            /*************************
             Fromato widescreen 16:9
             16 -> 100%
             9 -> 56.25%
            *************************/
            padding-bottom: 56.25%;
          }
          .responsive-media > * {
            position: absolute;
            width: 100%;
            height: 100%;
          }

Finalmente, podemos obtener el mismo resultado colocando la resolución de pantalla o aspect ratio que deseemos con una nueva técnica en CSS:

        
          <div class="aspect-ratio-16-9">
            <video src="/assets/media/sunset_evening_deck_877.mp4" controls></video>
          </div>
        
      
        
          .aspect-ratio-16-9 {
            /* aspect-ratio: 4/3; */
            /* border: thick solid coral; */
            aspect-ratio: 16/9;
          }
