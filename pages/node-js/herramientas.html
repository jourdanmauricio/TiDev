Herramientas para ser más felices: Nodemon y PM2

nodemon

Nodemon es una herramienta de código abierto que controla cualquier cambio en el código fuente y reinicia el proceso automáticamente.Es un demonio (daemon) que se encuentra corriendo y cada vez que detecte un cambio relanza la ejecución del proceso. 

        
          npm install -g nodemon

Cambie nodemon en lugar de nodo para ejecutar su código, y ahora su proceso se reiniciará automáticamente cuando cambie su código. Para instalar, obtenga Node.js, luego desde su terminal ejecute: 

        
          /* Ejecutamos con */
          nodemon variables_entorno.js

PM2

PM2 es similar a nodemon pero es más compleja y nos permite ejecución de aplicación en cluster, monitoreo de aplicaciones, mantener logs separados, relanza procesos ante cancelaciones, etc. No se utiliza en desarollo, es una herramienta para producción.

        
          npm install pm2 -g
        
      
        
          /* Ejecución */
          pm2 start variables_entorno.js
