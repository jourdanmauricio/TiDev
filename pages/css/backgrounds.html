Estilos background en CSS

La propiedad background es un shorthand de con muchas variaciones, por eso es conveniente no utilizarla el shorthand.

  • background-color
  • background-image: define la imagen de fondo del elemento
  • background-size: define el tamaño de la imagen de fondo, primer valor x, segundo y
    • cover: cambia el tamaño de la imagen de fondo para asegurarse de que permanezca completamente visible
    • contain: cambia el tamaño de la imagen de fondo para asegurarse de que el elemento esté completamente cubierto
  • background-repeat: define cómo se repite la imagen de fondo en el elemento
  • background-position: define la posición de la imagen de fondo, primer valor x, segundo y, si no se especifica un segundo valor éste será center aparte de valores numéricos podemos indicar el posicionamiento con las palabras: center, top, bottom, left and right
  • background-clip: define cuánto debe extenderse el fondo dentro del elemento.
  • background-origin: define el origen de la imagen de fondo.
  • background-attachment: define cómo se comportará la imagen de fondo al desplazarse por la página

background-color

Define el color de fondo del elemento

  • background-color: transparent; Por defecto, el color de fondo es transparente, significa que no hay color de fondo.
  • background-color: red; Podemos utilizar uno de los más de 140 nombres de color.
  • background-color: #05ffb0; Podemos utilizar códigos de colores hexadecimales.
  • background-color: rgb(50, 115, 220); Podemos utilizar códigos de colores RGB o RGBA
  • background-color: hsl(14, 100%, 53%); Podemos utilizar códigos de colores HSL o HSLA
          
            <div class="box bg-color"></div>
          
        
          
            .box {
              width: 300px;
              height: 300px;
              margin-left: auto;
              margin-right: auto;
              border: thick dashed red;
            }
            .bg-color {
              background-color: black;
            }

background-image

Define una imagen como fondo del elemento.

  • background-image: none; Elimina cualquier imagen de fondo.
  • background-image: url(/assets/images/jt.png); Utiliza la imagen definida en la ruta de la URL. Esta ruta puede ser relativa (al archivo css) o absoluta.
  • background-image: linear-gradient(red, blue); Podemos definir un degradado lineal como imagen de fondo. Necesitamos definir al menos dos colores. El primero comenzará en la parte superior, el segundo en la parte inferior. El ángulo predeterminado es hacia abajo (o 180 grados), lo que significa que el degradado es vertical, comenzando en la parte superior y terminando en la parte inferior del elemento.
  • background-image: linear-gradient(45deg, red, blue); Podemos especificar un ángulo, ya sea en grados o con palabras clave. Al usar grados, especifica la dirección del degradado.
  • background-image: radial-gradient(green, purple); Podemos definir un degradado radial como imagen de fondo. Necesitamos definir al menos dos colores. El primero estará en el centro, el segundo en los bordes.
  • background-image: radial-gradient(circle, green, purple); Podemos especificar la forma del degradado radial: círculo o elipse (predeterminado).
  • background-image: radial-gradient(circle, green 0%, purple 20%, orange 100%); Puede especificar paradas de color utilizando valores porcentuales.
  • background-image: radial-gradient(circle closest-side, green 0%, purple 20%, orange 100%); Puede especificar dónde debe terminar el degradado: closest-side, closest-corner, farthest-side, farthest-corner
  • background-image: radial-gradient(circle closest-side at 45px 45px, green 0%, purple 20%, orange 100%); Al igual que con la posición de fondo, podemos especificar la posición del degradado.
          
            <div class="box bg-image"></div>
          
        
          
            .bg-image {
              background-image: url("/assets/images/puesta-sol.jpg");
            }

La imagen posee una resolución muy grande para las dimensiones del contenedor. Luego volveremos sobre este punto.

background-size

Define el tamaño de la imagen de fondo.

  • background-size: auto auto; La imagen de fondo conservará su tamaño original. Si la imagen es más grande que su contenedor se recortará. Valor por defecto
  • background-size: 120px 80px; Podemos especificar un tamaño en píxeles: el primer valor es el tamaño horizontal el segundo es el tamaño vertical
  • background-size: 100% 50%; También podemos utilizar valores porcentuales. Esto puede alterar la relación de aspecto de la imagen de fondo y generar resultados inesperados
  • background-size: contain; La palabra clave constain cambiará el tamaño de la imagen de fondo para asegurarse de que permanezca completamente visible.
  • background-size: cover; La palabra clave cover cambiará el tamaño de la imagen de fondo para asegurarse de que el elemento esté completamente cubierto
          
            <div class="box bg-size"></div>
          
        
          
            .bg-size {
              background-image: url("/assets/images/puesta-sol.jpg");
              background-size: 300px 200px;
              background-size: 300px;
              background-size: cover;
              background-size: contain;
            }

background-repeat

Define cómo se repite la imagen de fondo en el fondo del elemento, comenzando desde la posición del fondo.

  • background-repeat: repeat; La imagen de fondo se repetirá tanto horizontal como verticalmente
  • background-repeat: repeat-x; La imagen de fondo solo se repetirá horizontalmente
  • background-repeat: repeat-y; La imagen de fondo solo se repetirá verticalmente
  • background-repeat: no-repeat; La imagen de fondo solo aparecerá una vez
          
            <div class="box bg-repeat"></div>
          
        
          
            .bg-size {
              background-image: url("/assets/images/puesta-sol.jpg");
              background-size: 200px;
              background-repeat: repeat;
              background-repeat: repeat-x;
              background-repeat: repeat-y;
              background-repeat: no-repeat;
            }

bg-position

Define la posición de la imagen de fondo.

  • background-position: 0% 0%; La imagen de fondo se colocará al 0% en el eje horizontal y al 0% en el eje vertical, lo que significa la esquina superior izquierda del elemento.
  • background-position: bottom right; Podemos utilizar una combinación de palabras clave de posición: centro, arriba, abajo, izquierda y derecha.
  • background-position: center center; La imagen de fondo se colocará en el centro del elemento.
          
            <div class="box bg-position"></div>
          
        
          
            .bg-position {
              background-color: skyblue;
              background-image: url("/assets/images/puesta-sol.jpg");
              background-size: 100px;
              background-repeat: no-repeat;
              background-position: 10px 20px;
              background-position: 1rem 2rem;
              background-position: 10% 20%;
              background-position: 10%;
              background-position: 10% center;
              background-position: top center;
              background-position: top right;
              background-position: top left;
              background-position: bottom center;
              background-position: bottom right;
              background-position: bottom left;
              background-position: bottom;
            }

background-clip

Define hasta dónde debe extenderse el fondo dentro del elemento.

  • background-clip: border-box; El fondo se extiende completamente por todo el elemento, incluso debajo del borde.
  • background-clip: padding-box; El fondo solo se extiende hasta el borde del borde: incluye el relleno pero no el borde.
  • background-clip: content-box; El fondo solo se extiende hasta el borde del contenido: no incluye el relleno ni el borde.
          
            <div class="box bg-clip"></div>
          
        
          
            .bg-clip {
              background-color: skyblue;
              background-image: url("/assets/images/puesta-sol.jpg");
              background-repeat: no-repeat;
              background-size: 200px;
              background-size: cover;
              background-clip: border-box; /* default */
              background-clip: padding-box;
              background-clip: content-box;
              padding: 1rem;
            }

background-origin

Define el origen de la imagen de fondo.

  • background-origin: padding-box; La imagen de fondo comienza en el borde del borde: dentro del relleno pero no en el borde
  • background-origin: border-box; La imagen de fondo comienza debajo del borde
  • background-origin: content-box; La imagen de fondo solo comienza en el borde del contenido: no incluye el relleno ni el borde
          
            <div class="box bg-origin"></div>
          
        
          
            .bg-origin {
              background-color: skyblue;
              background-image: url("/assets/images/puesta-sol.jpg");
              background-repeat: no-repeat;
              background-size: 200px;
              background-size: cover;
              background-origin: padding-box; /* default */
              background-origin: border-box;
              background-origin: content-box;
              padding: 1rem;
            }

background-attachment

Define cómo se comportará la imagen de fondo al desplazarse por la página.

  • background-attachment: scroll; La imagen de fondo se desplazará con la página. También se posicionará y cambiará de tamaño según el elemento al que se aplique
  • background-attachment: fixed; La imagen de fondo no se desplazará con la página y permanecerá posicionada de acuerdo con la ventana gráfica. También se posicionará y cambiará de tamaño de acuerdo con la ventana gráfica. Como resultado, la imagen de fondo probablemente solo sea parcialmente visible
          
            <div class="box bg-attachment"><
          
        
          
            .bg-attachment {
              background-color: skyblue;
              background-image: url("/assets/images/puesta-sol.jpg");
              background-repeat: no-repeat;
              background-size: cover;
              width: 100%;
              height: 50vh;
              background-attachment: scroll;
              background-attachment: fixed;
              /* background: red; */
            }
