Introducción a las Arquitectura CSS

La arquitectura CSS refiere a técnicas que nos ayudan a organizar y mantener nuestro código ordenado, óptimo y escalable.

Una buena arquitectua CSS debe ser

  • Predecible

    • Las reglas se comportarán como esperamos. Si agregamos una nueva característica esperamos que no afecte a otras secciones de nuestro sitio. Se encuentra relacionado al utility first, que nos permite afectar un atributo en particular como color de fondo, del texto, etc.

  • Reutilizable

    • Que se pueda utilizar en distintas secciones o elementos sin generar problemas.

  • Estable

    • Deberíamos poder modificar el código sin afectar a otras reglas.

  • Escalable

    • Cuando nuestro sitio crezca, el código tiene que seguir siendo fácil de mantener.

Comenzar con un plan nos pertirá cumplir con estas características evitándonos refactorizaciones futuras.

Una estrategía es comenzar definiendo clases utilizarias, las que solo aplican un estilo en particular. Esto nos generará elementos html textosos porque agregaremos varias clases a cada elemento, pero nos permite reutlizar el código entre distintintos proyectos.

Supongamos que creamos un primer proyecto con muchas clases utilitarias. Al momento de iniciar un segundo proyecto podemos copiarlas y agregar las nuevas clases utilitarias que necesitemos. Conforme crezcan nuestras clases utilitarias se transformarán en un miniframwork propio para maquetar proyectos.

De esta manera, diseñamos sistemas, no páginas en particular. Si diseñamos exclusivamente para el proyecto invertiremos demasiado tiempo en cada proyecto y no será reutilizable.

Para crear un flujo escalable, óptimo, escalable y ordenado tenemos que utilizar algunas herramientas como:

  • Metodologías: nos permiten dividir y nombrar las clases.
  • Frameworks: utilizamos estilos preexistentes para no comenzar desde 0.
  • Procesadores: trasnforman el código CSS para optimizarlo. Existen preprocesadores y postprocesadores.
  • Guías de estilo: cómo ordenamos y documentamos nuestros estilos. Ampliar el equipo de trabajo será mucho más facil y mentendremos las mismas normas sobre los estilos.
