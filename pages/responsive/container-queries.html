Container Queries

Los Container Queries nos permiten hacer componentes responsivos de interfaz de usuario, que no dependan del tamaño de la pantalla y las media queries para cambiar o adaptar el diseño de su contenido. Sino que dependan de sus propias necesidades de visualización. Hay quienes sostienen que será la evolución del responsive design.

Sintaxis

        
          .card {
            contain: layout inline-size style;
          }
          @container (min-width: 600px) {
            .card-container {
              display: flex;
            }
          }
