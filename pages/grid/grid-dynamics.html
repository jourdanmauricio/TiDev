Grids Dinámicas

La propiedad auto-fill rellena la grid con tracks generados dinámicamente, aunque tengamos menor cantidad de elementos los tracks se generan sin contenido. En cambio, la propiedad auto-fit ajusta la grid a los tracks existentes.

        
          .grid-dynamics {
            display: grid;
            /* Grid 4cols x ?  No definimos filas */
            height: 60vh;
            grid-template-columns: repeat(4, 100px);
            grid-template-columns: repeat(auto-fit, 100px);
            grid-template-columns: repeat(auto-fill, 100px);
            grid-template-columns: repeat(auto-fill, minmax(100px, 1fr));
            grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
          }
