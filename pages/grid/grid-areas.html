Grid con Áreas

Podemos asignar nombres de áreas.

        
          <section class="container-grid container-grid-areas grid-areas">
            <header class="item header">
              <p>Header</p>
            </header>
            <article class="item content">
              <p>Content</p>
            </article>
            <aside class="item sidebar">
              <p>SideBar</p>
            </aside>
            <footer class="item footer">
              <p>Footer</p>
            </footer>
          </section>        
        
      
        
          .container-grid-areas {
            height: 80vh;
          }
          .grid-areas {
            display: grid;
            /* Grid 2x4 */
            grid-template-columns: 1fr 200px;
            grid-template-rows: 100px 1fr 1fr 60px;
            grid-template-areas:
              "header header"
              "content sidebar"
              "content ."
              "footer footer";
          }
          .header {
            grid-area: header;
          }
          .content {
            grid-area: content;
          }
          .sidebar {
            grid-area: sidebar;
          }
          .footer {
            grid-area: footer;
          }
