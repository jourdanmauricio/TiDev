Ordenamiento de Grid Items

La propiedad order nos permita cambiar el orden en que se colocan los elementos del grid. El valor por default es 0 para todos los hijos y acepta números positivos y negativos. Mientras menor sea el número más al inicio se posicionará el elementos.

La porpiedad order aplica siempre que el posicionamiento se realice a través del flujo natural del grid. Si posicionamos explícitamente con nombres de áreas o a través de líneas, la propiedad no aplica. 
        
          <section class="container-grid grid-order">
            <article class="item">Grid Item 1</article>
            <article class="item">Grid Item 2</article>
            <article class="item">Grid Item 3</article>
            <article class="item">Grid Item 4</article>
            <article class="item">Grid Item 5</article>
          </section>
        
      
        
          .grid-order {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            grid-template-rows: repeat(2, 1fr);
            height: 30vh;
          }
          .grid-order .item:nth-child(1) {
            order: 1;
          }
          .grid-order .item:nth-child(2) {
            order: 2;
          }
          .grid-order .item:nth-child(5) {
            order: -1;
          }
