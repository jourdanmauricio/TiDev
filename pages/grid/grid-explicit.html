Si queremos posicionar un elemento en determinadas celdas podemos utilizar las propiedades grid-row-start/grid-row-end y grid-column-start/grid-column-end.

Colocaremos el elemento 10 ente las líneas 2 y 3, tanto en filas como en columnas. Donde inicialmente se encuentra el item 6, que será desplazado respetando el flujo.

A través de estas propiedades también podemos combinar celdas y crear áreas.

Otra forma es utilizar la palabra clave span.

        
          .grid-combination .item:nth-child(10) {
            color: red;
            grid-row-start: 2;
            grid-row-end: 3;
            grid-column-start: 2;
            grid-column-end: 3;
            /* Shortcut */
            grid-row: 2 / 3;
            grid-column: 2 / 3;
          }
          .grid-combination .item:nth-child(2) {
            background-color: cyan;
            /* Combinar celdas */
            grid-row: 1 / 2;
            grid-column: 3 / 5;
          }
          .grid-combination .item:nth-child(13) {
            background-color: orangered;
            /* Area - Shortcut de grid-row y grid-column */
            /* grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end; */
            grid-area: 3 / 4 / 5 / 6;
          }
          .grid-combination .item:nth-child(14) {
            background-color: greenyellow;
            /* Abarca 2 filas  */
            grid-row: span 2;
            grid-column: span 2;
          }
