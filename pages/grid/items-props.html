Propiedades de los items

Propiedades para elementos hijos

grid-column-start grid-column-end grid-row-start grid-row-end

Determina la ubicación de un elemento dentro de la cuadrícula haciendo referencia a líneas de cuadrícula específicas. grid-column-start/grid-row-start es la línea donde comienza el elemento, y grid-column-end/grid-row-end es la línea donde termina el elemento.

  • line: puede ser un número para hacer referencia a una línea de cuadrícula numerada o un nombre para hacer referencia a una línea de cuadrícula con nombre
  • span <number>: el elemento abarcará el número proporcionado de tracks de cuadrícula
  • span <name>: el elemento se extenderá hasta que llegue a la siguiente línea con el nombre proporcionado
  • auto: indica colocación automática, un intervalo automático o un intervalo default de uno
        
          .item-a {
            grid-column-start: 2;
            grid-column-end: five;
            grid-row-start: row1-start;
            grid-row-end: 3;
          }

Si no se declara ningún final de columna / final de fila, el elemento abarcará 1 track de forma predeterminada. Los elementos pueden superponerse entre sí. Puede usar z-index para controlar su orden de apilamiento.

grid-column grid-row

Abreviatura de grid-column-start + grid-column-end, y grid-row-start + grid-row-end, respectivamente.

        
          .item-c {
            grid-column: 3 / span 2;
            grid-row: third-line / 4;
          }

Si no se declara ningún valor de línea final, el elemento abarcará 1 pista de forma predeterminada.

grid-area

Asigna un nombre a un elemento para que pueda ser referenciado por una plantilla creada con la propiedad grid-template-areas. Alternativamente, esta propiedad se puede usar como una abreviatura aún más corta para grid-row-start + grid-column-start + grid-row-end + grid-column-end.

        
          .item {
            grid-area: <name> | <row-start> / <column-start> / <row-end> / <column-end>;
          }          

          .item-d {
            grid-area: header;
          }

justify-self

Alinea un elemento dentro de una celda a lo largo del eje en línea (fila). Este valor se aplica a un elemento dentro de una sola celda.

  • start: alinea los elementos para que queden al borde inicial de su celda.
  • end: alinea los elementos para que queden final de la celda.
  • center: alinea los elementos en el centro de su celda.
  • stretch: llena todo el ancho de la celda (este es el valor predeterminado).
        
          .item {
            justify-self: start | end | center | stretch;
          }

Se pueden alinear todos los elementos en el contenedor mediante la propiedad justify-items.

align-self

Alinea un elemento dentro de una celda a lo largo del eje de bloque (columna). Este valor se aplica al contenido dentro de un único elemento.

  • start: alinea los elementos para que queden al borde inicial de su celda.
  • end: alinea los elementos para que queden final de la celda.
  • center: alinea los elementos en el centro de su celda.
  • stretch: llena todo el alto de la celda (este es el valor predeterminado).
        
          .item {
            align-self: start | end | center | stretch;
          }

Se pueden alinear todos los elementos en el contenedor a través de la propiedad align-items.

place-self

Establece las propiedades align-self y justify-self en una sola declaración.

  • auto: la alineación "predeterminada" para el modo de layout.
  • align-self / justify-self: El primer valor establece align-self, el segundo valor justify-self. Si se omite el segundo valor, el primer valor se asigna a ambas propiedades.
    •           
            .item-a {
              place-self: center;
            }
