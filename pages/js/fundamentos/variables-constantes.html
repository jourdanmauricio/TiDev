Variables y constantes

Variables

Una variable es un espacio en la memoria con un nombre para guardar datos. Dentro de JavaScript tenemos tres formas de declarar una variable: var, const y let.

  • Var: Era la forma en que se declaraban las variables hasta ECMAScript 5. Casi no se usa porque su ámbito es global y posee las siguientes características:
    • Se puede reinicializar: el último dato predomina. 
                      
                  var music = "Pop";
                  ...  
                  var music = "Rock";
    • Se puede reasignar: a la variable ya inicializada le reasignamos otro valor. 
                      
                  var music = "Pop"
                  ... 
                  // no se incluye la palabra clave var
                  music = "Rock";
    • Su alcance es global: inicializamos la variable y la podemos invocar desde cualquier bloque de código pero no es recomendable utilizar VAR.

    • const y let es la forma en que se declaran las variables a partir de ECMAScript 6,

  • const: podemos declarar variables que nunca serán modificadas.
    • No se puede reinicilizar: no se puede redeclarar una const con el mismo nombre. 
                      
                  const music = "Pop"
                  ...
                  const music = "Rock" // Genera un error
    • No se pude reasignar: una vez que la hayamos inicializado no la podemos reasignar 
                      
                  const music = "Pop"
                  ... 
                  music = "Rock" // Genera un error
  • let: Son variables que pueden ser modificadas:
    • No se puede reinicilizar: no puede haber otra declaración con el mismo nombre. 
                      
                  let music = "Pop"
                  ...
                  let music = "Rock" // Genera un error
    • Se puede reasignar: a la variable ya inicializada le reasignamos otro valor. 
                      
                  let music = "Pop"
                  ...
                  music = "Rock"
    • Su contexto de es bloque: solo podemos utilizarla dentro de su bloque {}

Podemos guardar datos en ella utilizando el operador de asignación "=", y podemos resumir todas las sentencias en una sola línea.

        
          let message;

          let message2;
          message2 = 'Hola';

          let message3 = 'Hola';

También podemos declarar variables múltiples en una sola línea.

        
          let user = 'Mauro', age = 22, message = 'hola Mauro';

          // Para mejorar la legibilidad del código recomendamos utilizar varias líneas 
          
          let user = 'Mauro';
          let age = 22
          let message = 'hola Mauro';

Podemos cambiar el valor almacenado en la variable las veces que deseemos. De esta manera, eliminamos el valor anterior y almacenamos el nuevo.

        
          let message;
          message = 'Hola!';
          message = 'Mundo!'; // Nuevo valor

          alert(message);

También podemos declarar y copiar datos entre variables.

        
          let hello = 'Hola mundo!';
          let message;

          message = hello;

          alert(hello); // Hola mundo!
          alert(message); // Hola mundo!
Una variable se debe declarar solo una vez. Repetir la declaración de la misma variable provocará un error. Debemos declarar una variable una sola vez y desde entonces referirnos a ella sin let.

Nombre de las variables

  • El nombre puede incluir letras, dígitos, o los símbolos $ y _
  • El primer carácter no puede ser un dígito

Cuando el nombre contiene varias palabras, se utiliza camelCase. Las palabras van una detrás de otra, con cada palabra iniciando con letra mayúscula: nombreDeVariable.

Javascript diferencia entre mayúsculas y minúsculas. Ejemplos:

        
          // Nombres correctos
          let userName;
          let userEmail;
          
          // Nombres incorrectos
          let 1variable; // no puede iniciar con un dígito
          let my-name; // los guiones '-' no son permitidos en nombres

          // nombre y Nombre son variables distintas
          let nombre;
          let Nombre;

Ámbitos de declaración de variables:

  1. Ámbito global: es cuando se declara una variable fuera de una función, está disponible para cualquier código del documento en la que se declaró. Se declara con var.
  2. Ámbito de bloque: se utiliza para las variables declaradas con la palabra reservada let/const dentro de un bloque de código delimitados por llaves { }, por lo que no son accesibles fuera del bloque.
  3. Ámbito de función: cuando se declara una variable dentro de una función, se le denomina variable local, está disponible sólo dentro de la función en donde fue declarada. Se declarans con let/const.

Constantes

Cuando se sabe que una variable nunca cambiará de valor, puede declarar la variable con const. Las variables declaradas utilizando la palabra clave const se llaman “constantes”. No pueden ser alteradas. Al intentarlo causaría un error.

Se debe tener en cuenta que no se pueden alterar cuando el tipo de datos es primitivo (String, Number, etc). Cuando el tipo de dato es compuesto (objetos, arrays, etc) podemos alterar los valores porque el valor almacenado en la varible es la referencia al contenido, y no alteramos la referencia (alteramos el valor referenciado). 
          
            const banda = "La vela puerca";
            name = "La renga"; // Genera error
            ////
            const banda2 = {
              year = '1988',
              gender = 'Rock'
            }
            banda2.gender = 'Pop'
            // No genera error

Mayúsculas

En la práctica se utilizan los nombres de constantes con mayúsculas y guiones bajos cuando conocemos el valor previemente.

Existen constantes para las que no conocemos el valor inicialmente y será calculado en tiempo de ejecución, aunque el valor no cambiará después de la asignación. En este caso, no utilizamos la regla de las mayúsculas y las nombramos normalmente.

        
          const COLOR_RED = "#F00";
          const COLOR_GREEN = "#0F0";
          const COLOR_BLUE = "#00F";
          const COLOR_ORANGE = "#FF7F00";
          
          // ...cuando debemos elegir un color
          let color = COLOR_ORANGE;
          alert(color); // #FF7F00
