Expresiones regulares

Son secuencias de caracteres que forman un patrón de búsqueda generalmente utilizada para la búsqueda en cadenas de caracteres.

Documentación: wikipedia - mdn

Las expresiones regulares no son propias de JavaScript, se encuentran implementadas en muchos lenguajes de programación.

Uno de los usos más comunes son validaciones para el ingreso de datos, como una dirección de email (debería contener un signo @, solo contendrá numeros, letras y guiones, etc), un número de documento, números de tarjetas de crédito (16 dígitos, solo números).

No debemos aprender de memoria las reglas para expresiones regulares, muchas de ellas ya se encuentran construidas y podemos googlearlas. Si es importante entender cómo funcionan.

Crear una expresión regular

Existen dos formas para crear expresiones regulares, a través de la función constructora o a través de una expresión literal.

Las expresiones regulares literales proporcionan la compilación de la expresión regular cuando se carga el script. Si la expresión regular permanece constante, su uso puede mejorar el rendimiento.

El uso de la función constructora proporciona una compilación en tiempo de ejecución de la expresión regular. Usa la función constructora cuando sepas que el patrón de la expresión regular cambiará, o no conoces el patrón y lo obtienes de otra fuente, como la entrada del usuario.

Podemos utilizar flags como comodines en nuestra expresión regular. Ejemplo: 'g', busca todas las coincidencias, sino frenará en la primera.

Para validar la expresión regular existen varios métodos. Veremos test y exec. El método test retorna true o false, mientras que exec nos retorna más información como el texto de entrada, la posición donde ocurre la coincidencia, etc.

        
          let cadena = 'Lorem ipsum dolor sit amet consectetur adipisicing elit. 
          Iusto architecto ea voluptatum quaerat'. Lorem, quaerat"

          // Literal
          let re1 = /Lorem/;

          // Utilizando la función constructora
          let re2 = new RegExp('Lorem', 'gi');

          console.log(re1.test(cadena)); 
          // True
          console.log(re1.exec(cadena)); 
          // [0: "Lorem"
          // groups: undefined
          // index: 0
          // input: "Lorem ipsum dolor sit amet consectetur adipisicing elit. Iusto architecto ea voluptatum quaerat. Lorem, quaerat"
          // length: 1]
          
          let re3 = /\d/; // flag d -> números 0-9
          console.log( "re3", re3.test( cadena ) ); // false
          let re4 = /[0-9]/; // [] -> Carateres válidos 
          console.log( "re4", re4.test( cadena ) ); // false
