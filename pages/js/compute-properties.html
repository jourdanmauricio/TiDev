Propiedades dinámicas de los objetos

Supongamos que realizamos una consulta a una API que nos retorna objetos y deseamos crear una propiedad como el id dinámicamente. Existe un mecanismo para generar el nombre de la propiedad de forma dinámica.

En el ejemplo, a partir de un array de usuarios, agregaremos la propiedad id_numero, donde numero será la posición en el array.

        
          const objUsers = {};
          console.log( objUsers );
  
          const users = ['Mauricio', 'Isabel', 'Carlos', 'Hermes'];
  
          users.forEach( ( user, index ) => objUsers[`id_${index}`] = user );
          console.log( objUsers );
  
          console.log( "####################" );
  
          let aleatorio = Math.round( Math.random() * 100 + 5 );
          const objUsers2 = {
            [`id_${aleatorio}`]: "Valor aleatorio"
          };
  
          console.log( objUsers2 );
