JSON Server - API falsa local

Como aún no desarrollamos una aplicación backend que nos brinde una API utilizaremos una API falsa. Para ello deberemos instalar el paquete JSON Server.

Para utlizar JSON Server debemos tener instalado Node js.
Para verificar si tenemos instalado nodejs abrimos una consola y ejecutamos: node -v.

Instalación

        
          npm install -g json-server
