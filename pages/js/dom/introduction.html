Introducción al DOM

Web APIs

API (Application Programming Interfaces) es un término genérico que es usado para cubrir todos los requerimientos técnicos para tener varios componentes de software capaces de funcionar con los demás. En un contexto web, su uso es un poco más restrictivo, una API es usualmente un conjunto de métodos, propiedades y eventos para con el fin lograr ciertas tareas.

El objeto window representa la ventana que contiene un documento DOM; la propiedad document apunta al DOM document cargado en esa ventana.

El DOM es un modelo de documento que se carga en el navegador web y que representa el documento como un árbol de nodos, en donde cada nodo representa una parte del documento (puede tratarse de un elemento, una cadena de texto o un comentario).

El DOM es una de las APIs más usadas en la Web, pues permite ejecutar código en el navegador para acceder e interactuar con cualquier nodo del documento. Estos nodos pueden crearse, moverse o modificarse. Pueden añadirse a estos nodos manejadores de eventos (event listeners en inglés) que se ejecutarán/activarán cuando ocurra el evento indicado en este manejador.

El BOM (Browser Object Model) controla la ventana del navegador, barra de dirección, tamaño, posicionamiento, etc

El CSSOM (CSS Object Model) es un conjunto de APIs que permite manipular CSS desde JavaScript. Así como el DOM (Document Object Model) es para HTML, el CSSOM (CSS Object Model) es para CSS. Permite leer y modificar el estilo de CSS de forma dinámica.

Existen muchas Web APIs para realizar distintas tareas:

  • Eventos
  • Forms
  • AJAX - Fetch
  • History
  • Web Storage
  • Geolocation
  • Drag & Drop
  • Indexed DB
  • Canvas
  • MatchMedia
  • etc..

DOM

El objeto padre, el contexto global es window. De window cuelgan las APIs del navegador. Comenzaremos la del DOM, que es la que nos muestra el código html.

El Modelo de Objetos del Documento (DOM) es una interfaz de programación para documentos web. Representa la página para que los programas puedan cambiar la estructura, el estilo y el contenido del documento. El DOM representa el documento como nodos y objetos. Los lenguajes de programación pueden interactuar con la página.

Mostraremos por consola el objeto document. No es necesario colocar window.document porque window es el objeto global.

Podemos obtener los elementos del document como el head o el body accediendo a través de la notación de punto. Ejemplo: document.head, document.body. Tambien podemos acceder al juego de caracteres que utiliza el sitio (document.charset) o al title (document.title).

También podemos acceder colecciones de elementos. A priori, el resultado de los elementos que retornan colecciones parecen arreglos porque se encuentran entre paréntesis y poseen la propiedad length. Pero no son arreglos, no poseen todos los métodos de un array, no podríamos realizar un map. Tendríamos que almacenar el resultado en un arrego y luego aplicar el método map.

Ejemplos de colecciones:

  • lista de enlaces (document.links)
  • Imágenes a través de document.images
  • Formularios (document.forms)
  • Hojas de estilo enlazadas (document.styleSheets)
  • Scripts (document.scripts)

A través del método getSelection() podemos obtener la selección realiza en el documento.

Finalemente, aunque no es buena práctica, podemos escribir en el documento a través de document.write. Se agregará al final del documento.

        
          console.log( window.document );
          console.log( document );
          console.log( document.head );
          console.log( document.body );
          console.log( 'charset: ', document.charset );
          console.log( 'title: ', document.title );
          console.log( 'Links: ', document.links );
          console.log( 'Imagenes: ', document.images );
          console.log( 'Furmularios: ', document.forms );
          // Esperamos 2 segundos porque si recargamos la página perdemos la selección
          setTimeout( () => {
            console.log( 'Selección: ', document.getSelection().toString() );
          }, 2000
          );
          document.write('<h2>Desde el DOM</h2>');
