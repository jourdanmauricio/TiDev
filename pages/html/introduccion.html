HTML

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript). Manual de referencia

HTML no es un lenguaje de programación. Es un lenguaje de marcado que se utiliza para establecer la estructura y elementos de un sitio web como textos, hipervínculos, objetos, imágenes, etc. Los archivos HTML utilizan la extensión .htm o .html.

El archivo index.html es la página principal del dominio al cual se accede desde Internet, es el primero que vemos cuando acceder a un dominio en internet. Por ejemplo al acceder a www.dominio.com lo primero que intentará mostrar nuestro navegador es el archivo index

Estructura básica para páginas HTML

La forma más fácil de iniciar un documento HTML es utilizando el atajo '!' en Visual Studio Code, que incorpora por defecto el plugin emmet.


  <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta http-equiv="X-UA-Compatible" content="IE=edge">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Document</title>
    </head>
    <body>
    </body>
  </html>

La etiqueta <!DOCTYPE html> indica que utilizaremos HTML5 y <html lang="es"> indica el idioma que se utilizará en el contenido del documento. En este caso estamos especificando español. Es importante para posicionamiento de contenido.
