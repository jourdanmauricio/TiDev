Flexbox

Propiedades para el contendor padre

display

Define un contenedor flexible en línea o bloque dependiendo del valor dado. Permite un contexto flexible para todos sus hijos directos. Por default la alineación hacia afuera de la caja es block (valor flex). Si necesitamos colocar cajas una al lado de otra utilizamos display inline-flex.

        
          <section class="container-flex">
            <article class="item">Item 1</article>
            <article class="item">Item 2</article>
            <article class="item">Item 3</article>
          </section>
        
      
        
          .container-flex {
            display: flex; /* or inline-flex */
            background-color: #888;
          }
          .item {
            background-color: #666;
          }

flex-direction

Establece el eje principal, definiendo así la dirección en que se colocan los items en el contenedor. Flexbox es un concepto de diseño de una sola dirección.

        
          .container-flex-direction {
            /* flex-direction: row | row-reverse | column | column-reverse; */ 
            flex-direction: column-reverse;

flex-wrap

De forma predeterminada, todos los elementos flexibles intentarán encajar en una línea. A través de esta propiedad podemos cambiar este comportamiento permitiendo que los elementos se ajusten según sea necesario.

  • nowrap (predeterminado): todos los elementos flexibles estarán en una línea
  • wrap: los elementos flexibles se envolverán en varias líneas, de arriba a abajo
  • wrap-reverse: los elementos flexibles se ajustarán a varias líneas de abajo hacia arriba

El valor por defecto es nowrap, colocará los items en una sola línea. De acuerdo al tamaño de pantalla, si los elementos no entran generará una barra de scroll.

Si los elementos poseen un ancho definido se respetará siempre que entren en el contenedor. Si no entran el width será ignorado para intentar colocarlos en la misma línea.

flex-flow

Esta es una abreviatura de las propiedades flex-direction y flex-wrap, que juntas definen los ejes principal y transversal del contenedor flexible. El valor predeterminado es fila nowrap.

        
          .container {
            flex-flow: column wrap;
          }

justify-content

Define la alineación a lo largo del eje principal. Ayuda a distribuir el espacio libre adicional sobrante cuando todos los elementos flexibles de una línea son inflexibles o son flexibles pero han alcanzado su tamaño máximo.

También ejerce cierto control sobre la alineación de los elementos cuando desbordan la línea.

  • flex-start (default): los elementos se empaquetan hacia el inicio de la dirección flexible.
  • flex-end: los items se empaquetan hacia el final de la dirección flexible.
  • start: los elementos se empaquetan hacia el comienzo de la dirección del modo de escritura.
  • end: los elementos se empaquetan hacia el final de la dirección del modo de escritura.
  • left: los items se empaquetan hacia el borde izquierdo del contenedor, a menos que no tenga sentido con la dirección, entonces se comporta como start.
  • right: los items se empaquetan hacia el borde derecho del contenedor, a menos que no tenga sentido con la dirección flexible, entonces se comporta como un inicio.
  • center: los items están centrados a lo largo de la línea.
  • space-between: los items se distribuyen uniformemente en la línea; el primer elemento está en la línea de inicio, el último elemento en la línea final.
  • space-around: los items se distribuyen uniformemente en la línea con el mismo espacio alrededor de ellos. Tenga en cuenta que visualmente los espacios no son iguales, ya que todos los elementos tienen el mismo espacio en ambos lados. El primer elemento tendrá una unidad de espacio contra el borde del contenedor, pero dos unidades de espacio entre el siguiente elemento.
  • space-evenly: los elementos se distribuyen de modo que el espacio entre dos elementos cualesquiera (y el espacio hasta los bordes) sea igual.
        
          .container-flex-justify {
            /* justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly | start | end | left | right */
            justify-content: space-evenly;
          }

align-items

Define el comportamiento predeterminado de cómo se distribuyen los elementos flexibles a lo largo del eje transversal en la línea actual. Piense en ello como la versión de contenido justificado para el eje transversal (perpendicular al eje principal).

  • stretch (default): estirar para llenar el contenedor (respeta el ancho mínimo/ancho máximo).
  • flex-start / start / self-start: los elementos se colocan al inicio del eje transversal. La diferencia entre estos es sutil y se trata de respetar las reglas de dirección flexible o las reglas del modo de escritura.
  • flex-end / end / self-end: los elementos se colocan al final del eje transversal. La diferencia nuevamente es sutil y se trata de respetar las reglas de dirección flexible frente a las reglas del modo de escritura.
  • center: los elementos están centrados en el eje transversal.
  • baseline: los elementos están alineados según sus líneas base.
         
          .container-align {
            align-items: center;
          }

align-content

Alinea las líneas de un contenedor flexible dentro cuando hay espacio adicional en el eje transversal, de forma similar a como el contenido justificado alinea elementos individuales dentro del eje principal.

Nota: esta propiedad solo tiene efecto en contenedores flexibles de varias líneas, donde flex-wrap se establece en wrap o wrap-reverse). Un contenedor flexible de una sola línea (es decir, donde el ajuste flexible se establece en su valor predeterminado, sin ajuste) no reflejará el contenido de alineación.

  • normal (default): los items se colocan en su posición predeterminada como si no se hubiera establecido ningún valor.
  • flex-start / start: los items se colocan al inicio del contenedor. flex-start respeta la dirección, mientras que start respeta la dirección del modo de escritura.
  • flex-end / end: los items se colocan al final del contenedor. flex-end respeta la dirección, mientras que end respeta la dirección del modo de escritura.
  • center: items centrados en el contenedor.
  • space-between: items distribuidos uniformemente. La primera línea está al principio del contenedor mientras que la última está al final.
  • space-around: items distribuidos uniformemente con el mismo espacio alrededor de cada línea.
  • space-evenly: los elementos se distribuyen uniformemente con el mismo espacio a su alrededor.
  • stretch: las líneas se estiran para ocupar el espacio restante.
        
          <section class="container-flex container-flex-wrap container-align-content">
            <article class="item">NewItemExtend 1</article>
            <article class="item">NewItemExtend 2</article>
            <article class="item">NewItemExtend 3</article>
            <article class="item">NewItemExtend 4</article>
            <article class="item">NewItemExtend 5</article>
            <article class="item">NewItemExtend 6</article>
            <article class="item">NewItemExtend 7</article>
            <article class="item">NewItemExtend 8</article>
            <article class="item">NewItemExtend 9</article>
            <article class="item">NewItemExtend 10</article>
          </section>
        
      
        
          .container-align-content {
            align-content: space-between;
          }

gap, row-gap, column-gap

La propiedad gap controla el espacio entre elementos. El espacio se aplica solo a los elementos que no están en los bordes exteriores del contenedor.

        
          .container {
            display: flex;
            ...
            gap: 10px;
            gap: 10px 20px; /* row-gap column gap */
            row-gap: 10px;
            column-gap: 20px;
          }

El comportamiento podría considerarse como un canalón mínimo, como si el canalón fuera más grande de alguna manera (debido a algo como justificar contenido: espacio intermedio;) entonces la brecha solo tendrá efecto si ese espacio termina siendo más pequeño.
